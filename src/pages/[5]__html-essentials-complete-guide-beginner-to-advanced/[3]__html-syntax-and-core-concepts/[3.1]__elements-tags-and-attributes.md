## **Course:** HTML Essentials: Complete Guide (Beginner to Advanced)

**HTML basics**

- HTML stands for HyperText Markup Language.
- HTML is a markup language for structuring web pages.
- Browsers use HTML to display text, images, and other content.
- HTML uses **elements** and **attributes** to define page structure and behavior.

**Elements and tags**

- An HTML element wraps content.
- A typical element includes an opening tag, content, and a closing tag.
- Tags mark the start and end of an element.
- The content sits between the tags.
- HTML must be correctly nested.

**Attributes**

- Attributes add extra information to an element.
- Attributes appear in the opening tag.
- Attribute values are usually written in quotes.
- Attributes can change how an element behaves or appears.
- Use attributes when you need to describe or control an element more precisely.

**Void elements**

- Some HTML elements do not have closing tags.
- These are called **void elements**.
- Void elements stand alone and do not wrap content.
- Use them only where no closing tag is needed.

**Document structure**

- HTML documents need a doctype at the top.
- The doctype helps browsers interpret the document correctly.
- The `body` element contains the page content.
- Keep the document structure valid and properly nested.

**Practical writing rules**

- Use elements to enclose content you want browsers to treat as a unit.
- Use attributes to provide extra detail without changing the content itself.
- Keep tags balanced when the element requires a closing tag.
- Check nesting so elements stay in the correct order.

**Key terms**

- **Element**: the full HTML unit that wraps content.
- **Tag**: the markup that opens or closes an element.
- **Attribute**: extra information written in the opening tag.
- **Void element**: an element without a closing tag.

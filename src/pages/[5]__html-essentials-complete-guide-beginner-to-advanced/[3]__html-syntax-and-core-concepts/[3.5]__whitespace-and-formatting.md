## **Course:** HTML Essentials: Complete Guide (Beginner to Advanced)

**Whitespace in HTML**

- HTML is a markup language that structures content for browsers.
- Whitespace can affect how inline content renders.
- Spaces between inline elements may appear in the output.
- Correct nesting helps keep document structure predictable.

**What counts as whitespace**

- CSS whitespace processing treats spaces, tabs, line feeds, and carriage returns as document whitespace.
- `CR` characters are equivalent to spaces in this context.
- Non-breaking spaces (`&nbsp;`) are not part of that collapsing set.
- You can use whitespace in source code to improve readability.

**How CSS handles whitespace**

- The `white-space` property sets how whitespace inside an element is handled.
- It also controls whether lines wrap.
- Common values include `normal`, `pre`, `pre-wrap`, and `pre-line`.
- The `white-space-collapse` property controls whether whitespace sequences collapse or stay preserved.
- `white-space` and `text-wrap-mode` can be combined with the `white-space` shorthand.

**Preserving formatting with `<pre>`**

- The `<pre>` element presents text exactly as written in the HTML file.
- It usually renders text in a monospaced font.
- Leading newline characters immediately after the opening `<pre>` tag are handled specially by the parser.
- `<pre>` content is still parsed as HTML.
- Escape characters such as `<` when you need plain text inside `<pre>`.

**Practical formatting guidance**

- Use HTML comments and indentation for readability, but do not rely on source spacing for layout unless the element preserves it.
- Use `overflow-wrap`, `word-break`, or `hyphens` when you need words to break within themselves.
- Use `&nbsp;` when you need a space that does not collapse.
- Keep code examples consistent so whitespace differences do not distract from the lesson.

**Document structure reminders**

- Include the HTML5 doctype at the top of complete documents.
- Set the document language on the root element.
- Use snippets when a full document is not necessary.
- Keep block and inline behavior in mind when spacing content in normal flow.

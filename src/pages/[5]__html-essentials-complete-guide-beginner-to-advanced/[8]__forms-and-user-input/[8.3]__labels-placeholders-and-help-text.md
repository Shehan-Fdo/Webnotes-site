## **Course:** HTML Essentials: Complete Guide (Beginner to Advanced)

**Labels**

- Use the `<label>` element to provide a text description for a form control.
- Labels improve **accessibility** because screen readers announce them.
- Labels improve usability because clicking the label focuses the associated input.
- Labels stay visible after the input has a value.
- Labels increase the tap target on mobile devices.

**Connecting labels to inputs**

- Use `for` on `<label>` and match it to the input `id`.
- This explicit association is the recommended method.
- Keep the `id` value unique in the document.
- Use the same pattern for text inputs, email inputs, and checkboxes.

**Placeholders**

- Use the `placeholder` attribute to show a short hint when a control has no value.
- Use placeholders for an example value or a brief format hint.
- Do not use a placeholder instead of a `<label>`.
- Placeholder text disappears when the field has content, so it cannot replace a visible label.
- Supported HTML controls include `text`, `search`, `url`, `tel`, `email`, `password`, and `<textarea>`.
- Keep placeholder text short.
- Avoid line feeds in placeholder text outside `<textarea>`.

**Help text**

- Use help text to explain expected input or format.
- Use `aria-describedby` to connect a control to descriptive text.
- Use `aria-placeholder` only when you build a textbox with a non-semantic element.
- Keep help text separate from the label.
- Use help text for guidance, not for the main field name.

**Practical guidance**

- Put the label first in your form design.
- Add placeholder text only when it adds value.
- Use help text for extra context, examples, or format hints.
- Prefer visible text over hidden assumptions.
- Keep each field understandable without relying on placeholder text.

**Accessibility reminders**

- A control needs a descriptive name or label for assistive technologies.
- A label gives the primary name.
- A placeholder gives a temporary hint.
- A description gives extra support.
- Use the right element or attribute for each job.

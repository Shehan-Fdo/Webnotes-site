## **Course:** HTML Essentials: Complete Guide (Beginner to Advanced)

**Bold, italic, and emphasis in HTML**

- HTML uses text elements to show attention, importance, and emphasis.
- `b` draws attention to text without adding special importance.
- `strong` marks text with strong importance, seriousness, or urgency.
- `em` indicates emphasis and changes the meaning of the sentence by stressing a word or phrase.
- Browsers often render `b` and `strong` in bold.
- Browsers often render `em` in italics.
- Use semantic meaning first, not visual appearance alone.

**When to use `b` and `strong`**

- Use `b` for keywords, product names, or summary terms that need attention.
- Use `strong` for warnings, urgent notes, or content that matters to the page.
- Do not use `b` just to make text bold.
- Do not use `strong` just for styling.
- If you only want bold styling, use CSS `font-weight` instead.

**When to use `em`**

- Use `em` when you want to stress a word or phrase.
- Use `em` for emphasis that changes how the sentence reads.
- Keep emphasized text short and focused.
- Use `em` for meaning, not decoration.

**Practical usage notes**

- `b` only includes global attributes.
- `strong` only includes global attributes.
- `b` is useful for terms in summaries or labels.
- `strong` is useful for a single sentence or a few words when the message is important.
- `em` and `strong` support semantic HTML and accessibility.
- Clear semantics help browsers, search engines, and assistive tools understand content.

**Quick comparison**

| Element | Main purpose | Typical rendering | Best use |
|---|---|---|---|
| `b` | Draw attention | Bold | Keywords or terms without added importance |
| `strong` | Show strong importance | Bold | Warnings, urgency, or serious content |
| `em` | Add emphasis | Italic | Stressing a word or phrase in context |

**Good practice**

- Choose the element that matches the meaning.
- Keep emphasis minimal.
- Use CSS when you need presentation only.
- Use semantic elements when the text carries meaning.

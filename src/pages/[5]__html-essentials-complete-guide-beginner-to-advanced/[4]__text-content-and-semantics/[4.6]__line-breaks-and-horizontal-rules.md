## **Course:** HTML Essentials: Complete Guide (Beginner to Advanced)

**Line breaks with `<br>`**

- The `<br>` element inserts a line break in text.
- It is useful when line division matters, such as in poems or addresses.
- The text after `<br>` starts on the next line.
- Use `<br>` for intentional text breaks, not for spacing between paragraphs.
- `<br>` is a void element, so it does not wrap content.

**Horizontal rules with `<hr>`**

- The `<hr>` element creates a thematic break between sections.
- Use it to separate content blocks or indicate a shift in topic.
- Treat it as a structural divider, not a visual spacing tool.
- Keep the markup simple when you need a clear section break.

**Word break opportunities with `<wbr>`**

- The `<wbr>` element marks an optional word break opportunity.
- The browser may break the line there if needed.
- Use it for long words or strings that may overflow narrow layouts.
- On UTF-8 pages, `<wbr>` behaves like a zero-width space.
- It does not change bidi ordering in the cited example context.

**Practical usage**

- Use `<br>` for line-by-line text that must stay in one block.
- Use `<hr>` when you want a visible separation between ideas.
- Use `<wbr>` when you want the browser to decide where a long word may wrap.
- Choose the element that matches the meaning of the content.

**Accessibility and semantics**

- Prefer semantic structure over repeated break tags.
- Avoid using `<br>` to simulate paragraph spacing.
- Use the right element so the document stays easier to read and maintain.
- Keep line breaks meaningful to the content, not just the layout.

## **Course:** HTML Essentials: Complete Guide (Beginner to Advanced)

**Using HTML with JavaScript**

- JavaScript is a client-side programming language that makes webpages interactive.
- HTML provides the structure that JavaScript runs against in the browser.
- You can create useful webpages without JavaScript, but JavaScript adds more behavior.

**Adding JavaScript to HTML**

- Use the `<script>` element to embed executable code or reference JavaScript code.
- The `<script>` element can also hold data, not only JavaScript.
- The `async` attribute loads classic scripts in parallel and runs them when available.
- Do not use `async` on inline scripts without a `src` attribute.
- Keep script placement and loading behavior in mind when you want predictable execution.

**Events and user interaction**

- Events are signals the browser fires when something significant happens.
- Common triggers include clicks, window resizing, and other user actions.
- Use `addEventListener()` to set up event handlers.
- Use `removeEventListener()` when you need to stop listening.
- Inline event handler attributes exist, but avoid them for modern code.
- Event objects carry details about what happened.

**DOM scripting basics**

- The DOM (Document Object Model) is the browser’s internal representation of the HTML document.
- DOM scripting changes document structure through browser APIs.
- You can work with `Window`, `Document`, and `Navigator` in browser-based JavaScript.
- DOM nodes have relationships such as root, parent, child, sibling, and descendant.
- You can get references to nodes, create new nodes, and add or remove nodes and attributes.
- You can also change CSS styles with JavaScript.

**Practical HTML and JavaScript workflow**

1. Create a basic HTML document.
2. Add a `<script>` element or link to a JavaScript file.
3. Attach event handlers to elements that should respond to user actions.
4. Use DOM methods to update content, structure, or styles.
5. Test the page in a browser and confirm the interaction works as expected.

**Key takeaways**

- HTML and JavaScript work together to build interactive webpages.
- Events connect user actions to code execution.
- DOM manipulation lets you update the page after it loads.
- Use browser APIs and event listeners for cleaner, more maintainable behavior.

## **Course:** HTML Essentials: Complete Guide (Beginner to Advanced)

**Overview**

- HTML stands for HyperText Markup Language.
- HTML is a **markup language**, not a programming language.
- HTML gives a web page its structure and content.
- A personal profile page is a practical beginner project for learning core tags.

**Basic Page Skeleton**

- Start with `<!DOCTYPE html>` to declare HTML5.
- Use `<html>` as the container for the full page.
- Place settings and metadata in `<head>`.
- Put visible content in `<body>`.
- Use `<title>` to set the browser tab name.

**Core Content Elements**

- Use `<h1>` for the main page heading.
- Add `<p>` for short profile text.
- Use `<img>` to display a profile photo or other image.
- Use links and lists to organize contact details, interests, or skills.
- Keep the layout simple and readable.

**How HTML Works in a Profile Page**

- HTML acts like the **skeleton** of the page.
- Tags define what each part of the page means.
- Attributes add extra information to elements.
- Browsers read the structure and display the content.

**Beginner Build Approach**

1. Create the basic HTML document structure.
2. Add a page title in the `<head>`.
3. Add a welcome heading in the `<body>`.
4. Insert profile content such as text, images, and links.
5. Review the page in a browser and adjust the structure.

**Practical Notes**

- Start with simple content before adding more sections.
- Use clear headings to separate profile information.
- Build the page as a small project to practice HTML basics.
- Focus on structure first, then expand the design later.

## **Course:** HTML Essentials: Complete Guide (Beginner to Advanced)

**What alt text does**

- `alt` text provides a textual replacement for an image in HTML.
- Screen readers read `alt` text to help users understand image meaning.
- `alt` text also appears when an image fails to load.
- The `alt` attribute is mandatory on `<img>` elements.

**Why it matters for accessibility**

- Alt text helps users with visual disabilities access image content.
- Missing image descriptions can make content effectively invisible to assistive technology.
- Good HTML supports accessibility when you use the right element for the right job.
- Semantic HTML improves accessibility because browsers expose built-in accessibility hooks.

**How to write effective alt text**

- Describe the image’s purpose in context.
- Keep the text concise and useful.
- Match the description to what the image communicates.
- Use plain language when possible.
- Define abbreviations if you must use them.

**Writing process**

1. Identify the image’s role on the page.
2. Decide whether the image adds information, supports content, or is decorative.
3. Write a short description that matches the image’s function.
4. Test whether the text makes sense without seeing the image.
5. Revise if the description is vague, repetitive, or misleading.

**Common mistakes**

- Do not write alt text that only repeats nearby text.
- Do not leave meaningful images without a description.
- Do not add unnecessary detail that does not help the user.
- Do not use vague phrases that hide the image’s purpose.
- Do not treat alt text as a place for keyword stuffing.

**HTML implementation**

- Use the `alt` attribute on `<img>` to provide fallback text.
- Provide `src` or `srcset` so the image can load correctly.
- Keep the `alt` value aligned with the image’s purpose.
- Use the `alt` property in JavaScript when you set image text programmatically.

**Accessibility context**

- Alt text is one part of accessible web content.
- Keyboard accessibility, source order, and plain language also matter.
- CSS and JavaScript can help or harm accessibility depending on how you use them.
- Good focus styles and sensible hiding patterns support inclusive experiences.

**Standards and compliance**

- WCAG includes image text alternatives as an accessibility requirement.
- Accessibility guidance also appears in broader compliance efforts such as ADA and EAA contexts.
- Practical alt text writing should support both users and standards-based review.

**Quick checklist**

- Confirm the image needs a text alternative.
- Write for the image’s function, not just its appearance.
- Keep the description short and specific.
- Verify the text works with screen readers.
- Review the page for other accessibility issues too.

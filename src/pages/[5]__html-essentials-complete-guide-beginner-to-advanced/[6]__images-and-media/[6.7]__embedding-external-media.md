## **Course:** HTML Essentials: Complete Guide (Beginner to Advanced)

**Overview**

- HTML lets you structure content and embed media such as images, audio, video, and other external resources.
- Embedding adds richer content to a page without replacing the page itself.
- Choose the element that matches the content type and browser support you need.

**Core embedding elements**

- Use `<img>` for images.
- Use `<video>` for video playback.
- Use `<iframe>` to embed another HTML page in the current document.
- Use `<embed>` for external content from another application or source of interactive content.
- Use `<object>` for external resources such as PDF files and other embeddable content.

**`<img>` basics**

- Point `src` to the image resource.
- Add `width` and `height` to help avoid layout shifts while the image loads.
- Treat images as **replaced elements** when reasoning about how the browser renders them.
- Use `<figure>` when you need to pair an image with a caption.

**`<video>` basics**

- Use `<video>` to embed a media player for video playback.
- Include `controls` when you want the browser to show playback controls.
- Provide multiple `<source>` elements when you want fallback formats.
- Add `width` and `height` to control the display size.
- Use `autoplay` and `loop` only when they fit the user experience.

**`<iframe>` basics**

- Use `<iframe>` for a nested browsing context.
- Set `src` to the page you want to embed.
- Add a descriptive `title` for accessibility.
- Expect the embedded page to have its own document and navigation history.
- Use `width` and `height` to size the frame.

**`<embed>` and `<object>` considerations**

- Use `<embed>` only when you need external content from another application or source.
- Be cautious with `<embed>` because browser plug-in support is largely removed in modern browsers.
- Use absolute values for `<embed>` `height`; percentages are not allowed.
- Use `<object>` when you need a more general way to include external resources.

**Practical guidance**

- Keep media files small to improve loading performance.
- Check licensing before using media assets in a project.
- Use appropriately licensed files for images and other media.
- Prefer the simplest element that solves the embedding need.
- Test embedded content across browsers when support matters.

**Notes on browser behavior**

- Some embedding features are widely available across modern browsers.
- Support can still vary for specific parts of a feature.
- Avoid relying on deprecated plug-in behavior for production sites.

## **Course:** HTML Essentials: Complete Guide (Beginner to Advanced)

**What the `img` element does**

- The `img` element embeds an image into the document.
- It is a **void element**, so it has no content and no closing tag.
- It is widely supported across browsers.

**Basic syntax**

- Use `src` to point to the image resource.
- Provide `alt` for fallback text and accessibility.
- Include `width` and `height` to help avoid layout shifts while the image loads.
- Use at least one of `src` or `srcset` when embedding an image.

**The `alt` attribute**

- `alt` provides alternate text when the image does not display.
- Screen readers use `alt` to describe the image to users.
- The browser can show `alt` text if the image fails to load.
- The `alt` value should match the imageâ€™s purpose.

**Accessibility and fallback behavior**

- Images may fail because of network errors, content blocking, or broken links.
- Users may disable image loading.
- The image may still be loading when the page renders.
- Good `alt` text supports these cases.

**Captions with `figure` and `figcaption`**

- Use `figure` to group an image with related content.
- Use `figcaption` to add a caption or legend.
- `figcaption` gives the `figure` an accessible name.
- Keep the caption directly tied to the image content.

**Responsive and multi-format delivery**

- Use `srcset` when you need alternate image sources.
- Use `source` inside `picture` when you want to offer multiple formats.
- `source` is also a void element.
- Multiple formats help with browser compatibility.

**Performance and asset handling**

- Keep image files small for better web performance.
- Compress media assets when possible.
- Use lazy loading when it fits the page behavior.
- Smaller assets improve loading speed and user experience.

**Practical workflow**

1. Choose an appropriately licensed image.
2. Add the image with `img`.
3. Set `src` and `alt`.
4. Add `width` and `height` if you know them.
5. Group the image with `figure` and `figcaption` if you need a caption.
6. Test the page with images disabled or unavailable.

**Key takeaways**

- `img` embeds simple images in HTML.
- `alt` is essential for accessibility and fallback text.
- `figure` and `figcaption` add semantic captions.
- `srcset` and `source` help with flexible image delivery.
- Optimization matters for speed and user experience.

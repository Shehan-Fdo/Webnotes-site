## **Course:** HTML Essentials: Complete Guide (Beginner to Advanced)

**Link types for contact and files**

- Use the `<a>` element with `href` to create links to web pages, files, email addresses, and phone numbers.
- Put clear destination text inside the link so users know where it goes.
- Use `mailto:` for email links.
- Use `tel:` for phone links.
- Use a normal URL for web pages and downloadable files.

**Email links**

- Create an email link with `href="mailto:someone@example.com"`.
- Clicking a `mailto:` link opens the user's email application or chosen handler.
- Keep the visible text readable, such as an email address or a short label.
- Use contact details that match the intended recipient.

**Phone links**

- Create a phone link with `href="tel:+123456789"`.
- Use the full number format when possible so the link works consistently.
- Keep the visible text human-friendly, such as a formatted phone number.
- Use phone links for quick calling on supported devices.

**Download links**

- Use a file URL or path when linking to downloadable content.
- Add the `download` attribute when you want the browser to treat the target as a download.
- Use the link text to describe the file type or purpose.
- Make sure the linked file name and content are clear to the user.

**Contact information markup**

- Use `<address>` for contact information about a person or organization.
- Include the relevant name with the contact details.
- Place email and phone links inside `<address>` when they belong to that contact.
- Use `<address>` for contact context, not for arbitrary postal or layout content.

**Link behavior and accessibility**

- Links activate with the Enter key when focused and `href` is present.
- Write link text that describes the destination.
- Use link text that still makes sense out of context.
- Good link text helps accessibility and can support search clarity.

**Practical usage**

1. Choose the correct link type for the action you want.
2. Set `href` to `mailto:`, `tel:`, or the file path as needed.
3. Write clear link text.
4. Wrap related contact details in `<address>` when appropriate.
5. Test the link on the target device or browser.

**Key points**

- The `<a>` element handles email, phone, and file links.
- `mailto:` opens email handling.
- `tel:` opens phone handling.
- `<address>` groups contact information.

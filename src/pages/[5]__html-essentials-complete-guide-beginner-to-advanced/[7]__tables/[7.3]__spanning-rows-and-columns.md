## **Course:** HTML Essentials: Complete Guide (Beginner to Advanced)

**Table spans overview**

- HTML tables organize **tabular data** into rows and columns.
- Use tables for data, not for layout.
- A cell can span multiple columns with `colspan`.
- A cell can span multiple rows with `rowspan`.
- These attributes help group related values in a single table.

**`colspan`**

- `colspan` makes one cell occupy more than one column.
- It reflects the `colSpan` property in the DOM.
- The value is a positive number.
- A value of `0` is not the normal span count; browser behavior can treat it specially in the API.
- Use it when one heading or value covers adjacent columns.

**`rowspan`**

- `rowspan` makes one cell occupy more than one row.
- It reflects the `rowSpan` property in the DOM.
- The value is a positive number.
- Use it when one label or value applies across stacked rows.
- Keep the table structure consistent so other cells still align correctly.

**Practical use**

- Spanning cells can simplify schedules, price lists, and comparison tables.
- Use spans to show grouped data clearly.
- Keep tables simple when possible.
- Complex spanning can make tables harder to read and navigate.
- For accessibility, pair spans with clear headers and associations when needed.

**Accessibility notes**

- Use `th` for table headers.
- Use `scope`, `id`, and `headers` when you need stronger header-to-cell relationships.
- Add a caption when it helps explain the table.
- Group table content with `thead`, `tbody`, and `tfoot` when appropriate.
- Avoid overusing spans in dense data tables.

**Common table elements**

- `table` wraps the table.
- `tr` defines a row.
- `td` defines a data cell.
- `th` defines a header cell.
- `colgroup` and `col` help group columns for styling.

**Quick checks**

- Confirm the span value matches the number of rows or columns you want to cover.
- Check that the table still reads correctly after spanning cells.
- Verify that headers still describe the data clearly.
- Prefer the simplest structure that communicates the data well.

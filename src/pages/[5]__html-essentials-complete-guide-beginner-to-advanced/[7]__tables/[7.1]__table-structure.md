## **Course:** HTML Essentials: Complete Guide (Beginner to Advanced)

**Table purpose**

- Use tables for **tabular data** only.
- Use them to show information in rows and columns.
- Use them when you need quick lookup or comparison.
- Do not use tables for page layout.

**Basic table elements**

- Build a table with `<table>`.
- Use `<tr>` for a table row.
- Use `<td>` for a data cell.
- Use `<th>` for a header cell.

**Table headers**

- Use `<th>` to label rows or columns.
- Use headers to make table content easier to scan.
- Keep header text short and clear.

**Spanning cells**

- Use `colspan` to make a cell span multiple columns.
- Use `rowspan` to make a cell span multiple rows.
- Use spans when one value applies across more than one cell.

**Column grouping**

- Use `<colgroup>` to group columns.
- Use `<col>` to target individual columns.
- Use column grouping for styling purposes.

**Common uses**

- Use tables for schedules.
- Use tables for price lists.
- Use tables for product details.
- Use tables for other data that compares well across rows and columns.

**Content considerations**

- Tables can contain text, images, links, and other elements.
- Keep the structure readable before adding styling.
- Use CSS to improve design and readability.

**Practical approach**

1. Decide whether the content is tabular data.
2. Create the table structure with `<table>`, `<tr>`, and `<td>`.
3. Add `<th>` where headers improve clarity.
4. Add `colspan` or `rowspan` only when needed.
5. Group columns with `<colgroup>` and `<col>` if you need styling control.

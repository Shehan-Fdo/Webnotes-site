## **Course:** 200-301 CCNA

**Cisco DNA Center at a glance**

- Cisco DNA Center is Cisco’s network management platform for automation and assurance.
- Cisco now refers to the platform as **Cisco Catalyst Center** in current product materials.
- It uses AI to connect, secure, and automate network operations.
- It supports a single unified dashboard for network operations.

**Core capabilities**

- **Automation** helps you deploy networks faster.
- **Provisioning** and configuration automation reduce manual setup.
- **Policy automation** helps you apply consistent network behavior.
- **Segmentation** supports network separation and control.
- **Assurance** helps you monitor and troubleshoot network health.

**Operational value**

- It helps you scale the network with deeper insight into capacity and performance.
- It improves time to resolution with AI diagnostics and remediation.
- It supports better user experience through application insights.
- It helps optimize Wi-Fi coverage with 3D visualization.
- It can improve hybrid work experience by showing application behavior.

**Security and compliance**

- It supports **zero-trust** security goals for the workplace.
- It helps minimize exposure to security threats with an integrated security suite.
- It improves security posture through automated software and configuration management.
- It helps you maintain compliance with centralized control.

**Intent-based networking context**

- Cisco positions the platform as part of **intent-based networking**.
- Intent-based networking uses automation to align network behavior with business intent.
- Cisco describes automation as important for provisioning, orchestration, policy configuration, and issue resolution.
- The platform supports day-0 deployment and day-1, day-2, and ongoing operations.

**Assurance focus**

- Cisco Catalyst Assurance is part of the broader platform.
- Assurance helps you monitor overall enterprise health.
- It supports troubleshooting from a centralized view.
- Cisco documentation presents Assurance as a dedicated feature set within Catalyst Center.

**CCNA study focus**

- Know that Cisco DNA Center and Cisco Catalyst Center refer to the same platform family in current Cisco materials.
- Remember the main themes: automation, assurance, policy, segmentation, and centralized management.
- Expect the platform to appear in network automation and programmability topics.

## **Course:** 200-301 CCNA

**Dynamic routing basics**

- Dynamic routing uses **routing protocols** to exchange routing information between routers.
- Routers learn routes automatically instead of relying on manual path configuration.
- Dynamic routes are routes learned via routing protocols.
- You configure the networks to advertise on directly connected routers.
- Dynamic routing helps when the topology changes or links fail.

**Why dynamic routing matters**

- Static routing works well in small networks with stable topologies.
- Dynamic routing is more practical in large networks.
- Manual route changes take time when you have many routes.
- Frequent topology changes increase downtime if you use only static routes.
- Dynamic routing lets routers adjust the routing table based on network changes.

**How routers use routing protocols**

- Routers exchange route information with other interconnected routers.
- You still configure protocol parameters on the routers.
- You also establish routing and filtering policies.
- You verify that routers exchange information correctly.
- You troubleshoot when route exchange fails.

**Common protocol groupings**

- **Distance vector** protocols include RIP and IGRP.
- **Link state** protocols include OSPF and IS-IS.
- Cisco EIGRP is an advanced distance vector protocol.
- Some materials describe EIGRP as hybrid, but conservative study notes should treat it as advanced distance vector.
- **IGP** means interior gateway protocol.
- IGPs exchange routing information within one autonomous system.
- **BGP** is an exterior routing protocol.

**Routing concepts to remember**

- Routing selects and defines paths for IP packet traffic.
- Routers analyze data, choose routes, and forward traffic.
- Routing becomes more important as networks grow.
- Visibility into paths helps you identify latency sources.
- Cisco routing overview topics also include convergence, load balancing, route redistribution, and administrative distance.

**Static vs dynamic routing**

- Static routing requires the administrator to configure the path.
- Dynamic routing reduces manual configuration.
- Static routing can be efficient in small, stable networks.
- Dynamic routing scales better when routes change often.

**Study focus**

- Know what dynamic routing does.
- Know why it is useful in larger or changing networks.
- Know the basic protocol categories.
- Know that routers exchange routes through configured protocols.
- Know the difference between static and dynamic routing.

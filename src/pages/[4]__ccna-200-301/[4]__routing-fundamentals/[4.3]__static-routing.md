## **Course:** 200-301 CCNA

**Static routing overview**

- A **static route** is a route you configure manually.
- It is not learned through a dynamic routing protocol.
- Use static routing when you want direct control over path selection.
- Static routes are common in small or simple topologies.
- A default static route can send traffic to a next-hop gateway when no more specific route exists.

**Route types**

- A **default route** matches destinations not found in the local routing table.
- A **host route** points to one specific destination host.
- A **floating static route** uses a higher administrative distance than the preferred route.
- Floating static routes act as a backup path.
- You can use floating static routes alongside dynamic routing protocols or other static routes.

**Administrative distance**

- **Administrative distance (AD)** is the trust value used to choose between routes.
- Directly connected routes have an AD of `0`.
- Static routes normally have an AD of `1`.
- A floating static route uses a higher AD so another route stays preferred.
- Adjust AD when you want a static route to remain available but not active unless needed.

**Next-hop and exit-interface choices**

- You can configure a static route with a next-hop IP address.
- You can also configure a static route with an exit interface.
- Choose the form that fits the topology and forwarding behavior you want.
- A next-hop route points traffic to the next router.
- An exit-interface route points traffic out a specific interface.

**Practical use cases**

- Use a static default route on a remote office router with only one way out.
- Use a host route when you need a route for one device only.
- Use a floating static route for failover.
- Use static routing when you do not need a full routing protocol.

**Key takeaways**

- Static routes are manually configured.
- Default routes handle unknown destinations.
- Host routes target one destination.
- Floating static routes provide backup paths.
- AD determines which route the router prefers.

## **Course:** 200-301 CCNA

**Subnetting drill focus**

- Subnetting is a heavily tested CCNA skill.
- It supports IP address management and network segmentation.
- It helps you use IPv4 space efficiently.
- Practice improves speed and accuracy on exam questions.

**What you should calculate**

- Start with a target IP address and CIDR prefix.
- Find the **Network ID**.
- Find the **Broadcast IP**.
- Find the **First Host IP**.
- Find the **Last Host IP**.
- Find the **Next Network** address.

**How to approach practice**

1. Read the target IP and prefix carefully.
2. Identify the subnet boundaries.
3. Calculate the network and broadcast addresses.
4. Derive the first and last usable host addresses.
5. Confirm the next network address.

**Exam-style habits**

- Watch for wording like `fewest`, `least`, and `most`.
- Read each requirement carefully.
- Choose the answer that matches all constraints.
- Use quick mental calculation when possible.

**Practice workflow**

- Use repeated drills to build speed.
- Check your answers after each problem.
- Compare your result with the expected network values.
- Repeat until the calculations feel automatic.

**Helpful drill patterns**

- Work from random IP and CIDR combinations.
- Practice with different subnet sizes.
- Focus on the relationship between block size and address ranges.
- Use reference charts or calculators to verify results when needed.

**Common output fields**

- Network ID
- First Host IP
- Last Host IP
- Broadcast IP
- Next Network

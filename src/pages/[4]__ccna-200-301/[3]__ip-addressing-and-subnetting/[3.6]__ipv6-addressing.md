## **Course:** 200-301 CCNA

**IPv6 overview**

- IPv6 uses **128-bit** addresses.
- IPv6 addresses are written in **hexadecimal**.
- IPv6 expands the available address space far beyond IPv4.
- IPv6 supports unique addressing for a growing number of devices.

**Address structure**

- An IPv6 address includes a **prefix** and an interface portion.
- The prefix identifies the network part of the address.
- The remaining bits identify the host or interface.
- Prefix length is central to IPv6 subnetting.

**Prefix and subnetting**

- IPv6 subnetting focuses on the prefix length.
- You use the prefix to determine how much of the address belongs to the network.
- A shorter prefix leaves more bits for interface addressing.
- A longer prefix leaves fewer bits for interface addressing.

**Practical study points**

- Learn to recognize IPv6 addresses in hexadecimal form.
- Practice identifying the prefix length in address notation.
- Practice separating the network portion from the interface portion.
- Build comfort with IPv6 prefix and subnetting questions.

**Exam focus**

- Expect questions on IPv6 address format.
- Expect questions on prefix interpretation.
- Expect questions on subnetting concepts.
- Focus on understanding how IPv6 addresses are divided, not on memorizing unsupported details.

## **Course:** 200-301 CCNA

**IPv4 basics**

- IPv4 stands for Internet Protocol version 4.
- An IPv4 address is a **32-bit** string.
- You write IPv4 addresses in dotted decimal format.
- Each address has four decimal numbers separated by dots.
- Each number is an **octet**.
- An octet contains 8 bits.
- Each octet ranges from `0` to `255`.
- The full IPv4 range runs from `0.0.0.0` to `255.255.255.255`.
- IPv4 identifies a network interface on a device.

**Address structure**

- An IPv4 address has two parts.
- The **network part** identifies the network.
- The **host part** identifies the device on that network.
- The network part is on the left side of the address.
- The host part is on the right side of the address.
- A subnet mask tells you where the network part ends and the host part begins.

**Subnet mask role**

- A subnet mask separates network bits from host bits.
- A mask with more network bits leaves fewer host bits.
- A mask with fewer network bits leaves more host bits.
- The mask `255.255.255.0` uses 24 bits for the network portion.
- With that mask, the last 8 bits are for hosts.
- In `192.168.116.210/24`, the network address is `192.168.116.0`.
- In that same example, usable host addresses are `192.168.116.1` through `192.168.116.254`.

**Binary and decimal view**

- Computers use binary, not decimal, for address processing.
- Dotted decimal is the human-readable form of the binary address.
- Each octet maps to 8 binary bits.
- Binary helps you see how the subnet mask splits the address.

**Address classes and default masks**

- Subnet masks were used to group IPv4 addresses into classes.
- The common classes are A, B, and C.
- Each class has a default mask, also called a **natural mask**.
- Class A uses `255.0.0.0`.
- Class B uses `255.255.0.0`.
- Class C uses `255.255.255.0`.
- The default mask changes how many bits belong to the network and host portions.

**Why subnetting matters**

- Subnetting divides one larger network into smaller networks.
- Smaller networks can improve efficiency.
- Smaller networks can improve security.
- Smaller networks can make routing easier to read.
- Smaller networks can help conserve IPv4 addresses.

**Quick study points**

- Count IPv4 addresses as 32 bits total.
- Remember that each octet has 8 bits.
- Use the subnet mask to find the network and host portions.
- Convert between binary and dotted decimal when you analyze addresses.
- Check the mask first when you need the network address.

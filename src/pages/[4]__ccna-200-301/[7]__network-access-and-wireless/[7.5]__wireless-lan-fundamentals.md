## **Course:** 200-301 CCNA

**Wireless LAN overview**

- Wireless networking uses radio frequency (RF) signals instead of cables.
- Wireless LANs (WLANs) support mobility and flexible placement.
- Wired networks usually provide more predictable performance.
- Wireless networks are more exposed to interference and signal loss.
- Wireless design must account for coverage, capacity, and security.

**Wireless LAN topologies**

- A Basic Service Set (BSS) is a group of wireless clients connected through one access point.
- An Extended Service Set (ESS) links multiple BSSs to provide wider coverage.
- A mesh topology uses multiple wireless nodes to extend connectivity.
- Topology choice affects roaming and coverage area.
- Wireless bridging can connect wired devices to wireless networks.

**802.11 standards**

- IEEE 802.11 defines the wireless LAN family of standards.
- Standards determine how devices communicate over the air.
- Compatibility depends on supported 802.11 features.
- Channel and band selection affects performance and interference.
- Keep client and access point capabilities aligned when planning a WLAN.

**Traffic flow and communication types**

- Wireless traffic can move between clients and an access point.
- Traffic flow inside a BSS depends on the role of the access point.
- Some wireless designs support direct communication between devices.
- Coverage extension changes how traffic reaches distant clients.
- Understand the path traffic takes before you troubleshoot connectivity.

**Coverage extension and bridging**

- Use wireless bridging to connect separate network segments.
- A universal workgroup bridge can attach wired devices to a wireless network.
- Point-to-point bridges connect two locations.
- Point-to-multipoint bridges connect one site to several remote sites.
- Choose the bridge type based on distance and site layout.

**Wireless security concepts**

- Wireless security protects data sent over shared radio channels.
- Authentication verifies a client before access.
- Encryption helps protect wireless traffic from interception.
- IEEE 802.1X is a common authentication framework in enterprise WLANs.
- Extensible Authentication Protocol (EAP) supports multiple authentication methods.
- Fast roaming and session resumption help maintain connectivity during movement.

**Wireless performance and interference**

- RF signals can weaken as distance increases.
- Interference can reduce throughput and reliability.
- Channel management helps reduce overlap and contention.
- Environmental conditions affect wireless performance.
- Plan for signal quality, not just signal presence.

**Wireless network trends**

- Wireless networks support modern mobile and distributed work.
- WLANs are common in offices, classrooms, hospitals, and public spaces.
- Reliable wireless access is often critical to daily operations.
- Scalability matters when user density grows.
- Design for both current demand and future expansion.

**Exam focus**

- Know the difference between wired and wireless behavior.
- Recognize common WLAN topologies.
- Understand the role of 802.11 standards.
- Identify basic wireless security concepts.
- Be able to explain how coverage is extended.

## **Course:** 200-301 CCNA

**OSI model**

- The **OSI** (Open Systems Interconnection) model is a reference model for describing network communication.
- It helps you understand how data moves across a network.
- It has seven layers.
- The layers describe different functions of data traveling through the network.
- The model provides a framework for networking standards and devices.

**TCP/IP model**

- The **TCP/IP** model is another widely used network model.
- It is used to describe how communication happens across networks.
- It is a common comparison point with the OSI model in CCNA study.
- The evidence emphasizes understanding both models as core network fundamentals.

**Why these models matter**

- These models help you understand how network applications communicate through network media.
- They give you a structured way to study network functions.
- They support troubleshooting and network design by separating communication into layers.
- Cisco expects you to know core network fundamentals for CCNA.

**Layer study tips**

- Learn the layer names in order.
- Remember that the Physical layer is often treated as the first layer when numbered from the bottom.
- Use a mnemonic if it helps you recall the OSI layers.
- Focus on the function of each layer, not just the names.

**CCNA exam focus**

- Expect questions about the OSI and TCP/IP models.
- Know that the OSI model has seven layers.
- Know that the models are used to explain how data travels through a network.
- Use the models as a foundation for later networking topics.

## **Course:** 200-301 CCNA

**Overview**

- `CCNA` stands for Cisco Certified Network Associate.
- The `200-301 CCNA` exam is the current CCNA exam.
- Cisco says the exam tests knowledge and skills in:
  - network fundamentals
  - network access
  - IP connectivity
  - IP services
  - security fundamentals
  - automation and programmability
- Passing the exam earns the `CCNA` certification.
- Cisco lists the exam duration as `120 minutes`.
- Cisco lists the exam price as `$US300`, or you can use Cisco Learning Credits.

**Introduction to Networks**

- `Introduction to Networks` is the first CCNA learning area.
- Cisco Networking Academy offers `CCNA: Introduction to Networks`.
- This topic builds the base for later CCNA networking study.
- Focus on core networking concepts before moving to switching, routing, and wireless topics.

**What You Need to Know**

- Learn how networks support communication between devices.
- Learn the basic purpose of network devices and services.
- Learn the exam language Cisco uses for the CCNA blueprint.
- Learn the difference between broad exam domains and specific course topics.
- Use the official exam topics as your reference point when studying.

**Study Focus**

- Review `network fundamentals` first.
- Connect each concept to a practical networking use case.
- Keep track of terms Cisco uses in the exam overview.
- Build a habit of matching study notes to the exam domains.
- Treat this topic as the foundation for the rest of the CCNA path.

**Exam Context**

- Cisco positions CCNA as a certification for networking professionals.
- The exam supports roles such as:
  - network engineer
  - systems administrator
  - network administrator
  - IT consultant
- The certification is intended to validate practical networking knowledge.
- Cisco also notes that the exam can count toward recertification goals.

> Use the official Cisco exam topics when you study, because they define the scope more reliably than third-party summaries.

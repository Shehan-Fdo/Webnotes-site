## **Course:** CompTIA Network+ (Exam N10-009)

**Fiber optic basics**

Fiber optic cable carries data as pulses of light rather than electrical signals. In networking, that light is generated by equipment such as lasers and travels through the fiber to represent digital ones and zeros. This makes fiber a strong fit for high-speed links and long-distance transport where copper would lose signal quality more quickly.

- Fiber uses **light pulses** to transmit data.
- The signal path depends on the fiber type and its core size.
- Fiber is commonly used where speed and distance matter.

**Single-mode fiber**

Single-mode fiber has a very small core, typically about 9 µm (micrometers) in diameter. Its narrow pathway allows only one mode of light to travel through the cable at a time. Because the light follows a single path, single-mode fiber is suited to longer runs and higher-distance network links.

- **Core diameter** is small, around 9 µm.
- Light travels in one mode through the fiber.
- Use it when you need long-distance transmission.

**Multimode fiber**

Multimode fiber has a larger core, usually 50 µm or 62.5 µm. The wider core allows multiple modes of light to travel through the cable, meaning the signal can take different paths from one end to the other. This design supports shorter-distance networking and is commonly contrasted with single-mode fiber in cable selection.

- **Core diameter** is larger, usually 50 µm or 62.5 µm.
- Light can travel in multiple modes.
- Use it for shorter links where the design fits the application.

**Comparing the two fiber types**

The main difference between single-mode and multimode fiber is how the core guides light. Single-mode uses one path through a smaller core, while multimode uses multiple paths through a larger core. When you choose between them, match the cable type to the network’s distance and performance needs.

| Feature | Single-mode | Multimode |
|---|---|---|
| Core size | About 9 µm | 50 µm or 62.5 µm |
| Light path | One mode | Multiple modes |
| Typical fit | Longer distance | Shorter distance |

> Use the correct fiber type for the connection. A mismatch can cause the link to behave incorrectly because light travels differently in each type.

**Troubleshooting focus**

When you troubleshoot fiber, verify that the installed cable type matches the expected connection. Fiber mismatches are a physical-layer issue, so the link may fail even when the rest of the network is configured correctly. If the connection is unstable or does not come up, confirm the fiber type before moving to higher-layer causes.

- Check whether the link expects single-mode or multimode.
- Confirm the core size and cable type match both ends.
- Treat fiber mismatch as a physical-layer problem first.

**Exam takeaway**

For Network+ purposes, remember the core distinction: single-mode has a smaller core and one light path, while multimode has a larger core and multiple light paths. That difference drives distance, application fit, and troubleshooting checks.

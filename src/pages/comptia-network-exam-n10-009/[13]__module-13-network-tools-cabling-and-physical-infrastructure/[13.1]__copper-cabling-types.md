## **Course:** CompTIA Network+ (Exam N10-009)

**Copper cabling overview**

Copper cabling is a core part of wired networking, and the cable you choose affects throughput, distance, and installation fit. In practice, you match the cable type to the network’s current needs and expected growth, because cabling is difficult to replace once installed.

- **Twisted pair** is the most common copper media for Ethernet.
- Copper cabling supports wired links for routers, switches, wireless access points, and other infrastructure devices.
- Cable selection matters because the network gear may define speed, but the cabling must support it.

**Twisted pair fundamentals**

Twisted pair copper cable usually contains multiple wires twisted together inside one sheath. Ethernet LANs commonly use unshielded twisted pair, or <abbr title="Unshielded Twisted Pair">UTP</abbr>, which uses 8 copper wires arranged as four twisted pairs.

The twisting helps the cable carry signals reliably, and the pairs are terminated according to standard pinouts. Typical Ethernet UTP uses RJ-45 connectors on both ends, with eight pins that accept the eight conductors.

- UTP cable uses 8 wires in 4 twisted pairs.
- Each pair has a color scheme with one solid wire and one striped wire.
- RJ-45 connectors are the common termination for Ethernet copper cabling.

**Cable categories: Cat5e, Cat6, and Cat6a**

Category cables differ in capability, and the main families highlighted in the evidence are Cat5e, Cat6, and Cat6a. These categories are used to support different throughput needs and future planning.

Cat5e uses four wire pairs to deliver gigabit Ethernet. Cat6 and Cat6a are also common choices, with Cat6a generally positioned for stronger support of higher-speed networking and longer-term installations.

| Category | Practical note from the evidence |
|---|---|
| Cat5e | Uses four wire pairs to deliver gigabit Ethernet |
| Cat6 | Common copper category for Ethernet deployments |
| Cat6a | Common copper category with stronger high-speed support |

- **Cat5e** is a baseline choice for gigabit Ethernet.
- **Cat6** is a common upgrade path when you want more headroom.
- **Cat6a** is often chosen when you want better support for future bandwidth needs.

**Straight-through and crossover cabling**

Different devices may transmit and receive on different pin pairs, so the cable type must match the connection. The evidence identifies two main Ethernet cable types: straight-through and crossover.

A straight-through cable keeps the same pinout on both ends. A crossover cable swaps the transmit and receive pairs so devices can communicate correctly when their pin usage differs.

- **Straight-through** cables use the same wiring pattern on both ends.
- **Crossover** cables swap transmit and receive pairs.
- Correct pin mapping matters when connecting devices with different transmit/receive layouts.

**Installation and selection considerations**

Copper cabling is not just about speed. You also consider installation environment, cable rating, and whether the cable can support the network’s expected use. The evidence notes plenum-rated copper cables as an important topic, which means cable jacket choice can matter in building spaces.

You should also think about cost and future replacement risk. A cable that is adequate today may become a bottleneck later if the network grows or if higher throughput becomes necessary.

> Once copper cabling is installed, replacing it is difficult and disruptive, so choose carefully the first time.

**Exam-focused takeaways**

For Network+ study, focus on the relationship between cable type, connector type, and supported performance.

1. Identify twisted pair copper as the standard Ethernet medium.
2. Recognize UTP as 8 wires in 4 twisted pairs with RJ-45 connectors.
3. Distinguish Cat5e, Cat6, and Cat6a by their role in supporting Ethernet speeds.
4. Know when straight-through versus crossover cabling is appropriate.
5. Remember that installation constraints and future needs affect cable choice.

**Quick memory cues**

- **Copper cabling** is foundational for wired network infrastructure.
- **UTP** is the common twisted pair format for Ethernet.
- **RJ-45** is the standard connector you expect on Ethernet copper cables.
- **Category cables** help you match performance to the network design.
- **Cable choice** should account for both current throughput and future growth.

## **Course:** CompTIA A+ (Exams 220-1201 & 220-1202)

**Software installation on macOS**

CompTIA A+ expects you to recognize common macOS installation formats and the basic flow of installing software. macOS uses disk images and installer packages to distribute applications, and both are common in support scenarios.

- A **DMG** file is an Apple disk image.
- When you double-click a DMG, it opens in Finder and shows the contents of the image.
- Developers often use DMG files to distribute software before you launch the installer inside.
- A **PKG** file is an installer package.
- A PKG behaves like a Windows `setup.exe` by starting an installation workflow and prompting you for required choices.

> macOS software distribution is often tied to Apple-managed formats, so you should identify the file type before you begin troubleshooting an install.

**Managing updates and system support**

The exam also emphasizes operating system management tools that support maintenance and recovery. On macOS, backups and disk utilities are part of that support workflow.

- **Time Machine** is the built-in macOS backup tool.
- It keeps hourly backups for the last 24 hours.
- It keeps daily backups for the last month.
- It then continues with weekly backups as space allows.
- When the backup drive fills, Time Machine removes the oldest data as new backups are created.
- **Disk Utility** is used for disk maintenance when needed.
- Disk Utility is not typically a scheduled task.
- The **First Aid** function in Disk Utility helps address drive-related problems.

**Practical installation and recovery workflow**

When you support software installation or updates, you usually verify the package type first, then use the appropriate installer or maintenance tool. This keeps the process predictable and helps you separate installation issues from storage or disk issues.

1. Identify whether the software arrives as a DMG or PKG.
2. Open the DMG in Finder or launch the PKG installer.
3. Complete the installation prompts as required.
4. If the system has storage or disk problems, use Disk Utility and its First Aid option.
5. Use Time Machine backups when you need to recover data after a failed update or other system issue.

**Key exam takeaways**

- macOS software commonly arrives as **DMG** or **PKG** files.
- DMG files expose the contents of a disk image.
- PKG files run an installation process.
- Time Machine provides rolling backups for recent recovery points.
- Disk Utility supports disk maintenance and repair tasks.

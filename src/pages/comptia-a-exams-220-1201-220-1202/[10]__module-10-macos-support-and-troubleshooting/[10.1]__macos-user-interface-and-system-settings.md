## **Course:** CompTIA A+ (Exams 220-1201 & 220-1202)

**macOS configuration overview**

macOS uses **System Preferences** in older versions and **System Settings** in newer versions as the central place to change operating system behavior. It serves a similar role to the Windows Control Panel or Settings app, so you use it when you need to adjust how the Mac looks, connects, protects data, or supports users.

You can open the settings area from the **Apple menu** or by using **Spotlight** search. For A+ exam purposes, focus on the fact that this is the main hub for common administrative and user-facing changes.

- Display layout and resolution
- Network configuration
- Printer setup
- Privacy and security permissions
- Accessibility options
- Backup settings

**Display and monitor settings**

The **Displays** area controls how the screen and connected monitors behave. This is where you adjust resolution, brightness, color-related options, and multi-monitor arrangement.

When multiple displays are connected, you can position them to match the physical layout on the desk. You can also choose which display shows the menu bar and primary interface elements.

- **Resolution** changes the clarity and size of text and icons.
- **Arrangement** controls side-by-side or mirrored monitor layouts.
- **Brightness** changes screen light level.
- **Color profiles** help keep colors accurate.
- **Night Shift** reduces blue light for user comfort.

**Network, printers, and user support**

System settings also centralize common support tasks such as network access and peripheral setup. If a user cannot connect, print, or reach a shared resource, this is one of the first places you check.

The same area also supports user-focused configuration, including accessibility features and privacy controls. These settings help tailor the Mac to the user while also limiting which apps and services can access system resources.

- **Network** settings manage connectivity and adapter behavior.
- **Printers & Scanners** adds and maintains printing devices.
- **Privacy** controls permissions and access to sensitive features.
- **Accessibility** supports users with different interaction needs.

**Backup and recovery tools**

macOS includes **Time Machine**, a built-in backup utility. It creates rolling backups so you can restore data from recent points in time. The backup history includes hourly backups for the last 24 hours, daily backups for the last month, and weekly backups after that.

As backups continue, Time Machine keeps using the backup drive until it fills up, then it removes the oldest data as new backups are created. This makes it useful for ongoing recovery rather than one-time archival storage.

**Disk maintenance**

For drive-related maintenance, macOS uses **Disk Utility**. This tool is used as needed rather than on a fixed schedule, especially when a drive shows signs of trouble.

A common maintenance action is **First Aid**, which checks and repairs disk-related issues in a way that is similar in purpose to repair tools on other operating systems.

> Use Disk Utility when you suspect storage problems, but treat it as a maintenance tool rather than a routine daily task.

**System Settings changes in newer macOS versions**

Newer macOS releases streamline the settings layout and rename some areas. For example, **Apple ID** becomes **Apple Account** in macOS Sequoia, and system extension management moves under `General > Login Items & Extensions`.

This matters for support work because the feature is still present, but the path to it changes. When you troubleshoot, confirm the macOS version before assuming a setting is in the same place as an older release.

**Quick exam reminders**

- macOS settings are the central place for configuration changes.
- System Preferences and System Settings refer to the same general management area across versions.
- Displays, network, printers, privacy, accessibility, and backup tools are common exam targets.
- Time Machine provides rolling backups.
- Disk Utility handles disk maintenance and First Aid repair tasks.

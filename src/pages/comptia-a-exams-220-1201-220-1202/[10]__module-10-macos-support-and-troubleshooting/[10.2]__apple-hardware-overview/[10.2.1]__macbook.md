## **Course:** CompTIA A+ (Exams 220-1201 & 220-1202)

**MacBook and macOS overview**

A MacBook runs **macOS**, which uses familiar operating-system concepts such as file types, system settings, and backup tools. For A+ exam prep, focus on the built-in utilities and where you would use them during normal support tasks.

macOS includes several management features that help you configure the system, maintain storage, and recover data when needed. The most useful topics here are the System Preferences app, Time Machine, Disk Utility, and common macOS file extensions.

- **System Preferences** is the main place to change macOS configuration.
- It is similar in purpose to the Windows Control Panel.
- You can use it for settings such as displays, network, printers and scanners, privacy, accessibility, users and groups, and Time Machine.
- You can open it from the Apple menu or search for it with Spotlight.

**System Preferences and display configuration**

When you need to adjust a MacBook’s behavior, System Preferences is the first place to check. It centralizes many common settings, so you do not need to hunt through separate apps for basic configuration changes.

Display settings are a common example. You can change monitor layout, choose which display shows the Apple menu, and adjust resolution. This matters when a MacBook is connected to external monitors or used in a multi-display setup.

- Use **Displays** to arrange screens left to right.
- Change the active display for the Apple menu when multiple monitors are connected.
- Modify resolution from the same settings area.
- Check network and iCloud-related settings in System Preferences when configuration changes are needed.

**macOS file types**

macOS uses file extensions to indicate file purpose, just like other operating systems. Knowing the common extensions helps you identify installers and disk images quickly.

- `.dmg` is an **Apple disk image**.
- A `.dmg` file is similar to an ISO image and mounts like a drive in Finder.
- `.pkg` is an **installer package** used to install software.
- `.zip` is a compressed archive format you may also see on macOS.

These file types matter because they affect how you open, mount, or install software on a MacBook. If you see a `.dmg`, expect to mount it first; if you see a `.pkg`, expect an installer workflow.

**Backup and recovery with Time Machine**

**Time Machine** is the built-in macOS backup tool. It creates rolling backups so you can recover older versions of files or restore a system state from earlier points in time.

The backup pattern is straightforward:
- Hourly backups for the last 24 hours.
- Daily backups for the last month.
- Weekly backups after that.

Time Machine continues backing up until the backup drive fills up. After that, it removes the oldest backups as new ones are created. This makes it useful for ongoing protection without manual cleanup.

> Time Machine is a backup feature, not a disk repair tool. Use it for recovery and version history, not for fixing drive errors.

**Disk maintenance with Disk Utility**

If a MacBook has storage problems, use **Disk Utility** for maintenance tasks. It is not typically something you schedule routinely; you use it when a drive needs attention.

A key feature is **First Aid**, which checks and repairs disk-related issues. This is the macOS equivalent of a basic disk repair workflow you would use when troubleshooting storage problems.

- Open Disk Utility when you suspect drive issues.
- Use First Aid to check the disk.
- Treat it as an on-demand maintenance tool rather than a regular scheduled task.

**Practical exam focus**

For A+ questions, connect each tool to its purpose:
- **System Preferences** for configuration changes.
- **Time Machine** for backups and recovery.
- **Disk Utility** for disk maintenance.
- `.dmg` and `.pkg` for software installation workflows.

A MacBook is not just “a laptop running Apple software”; it is a system with specific built-in tools that support setup, backup, and troubleshooting. If you can match the tool to the task, you cover the most testable part of this topic.

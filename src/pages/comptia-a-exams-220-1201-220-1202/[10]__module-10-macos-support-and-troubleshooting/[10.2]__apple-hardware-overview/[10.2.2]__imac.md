## **Course:** CompTIA A+ (Exams 220-1201 & 220-1202)

**iMac overview**

The iMac is Apple’s all-in-one desktop computer line. It combines the display and core computer hardware in a single enclosure, which makes it a compact desktop option for macOS users. In the Intel era, iMac models used Intel-based chipsets, and the line later transitioned to Apple silicon.

- **All-in-one** design keeps the computer and display together.
- Intel-based iMacs were sold from 2006 to 2022.
- The iMac sits in the desktop lineup as an alternative to the Mac mini and below the Mac Pro in performance range.
- Apple also sold the iMac Pro alongside the standard iMac for a period.

**macOS file types and installation media**

macOS uses file extensions to identify file types, and you should recognize the common installation formats. A `.dmg` file is an Apple disk image. When you open it, Finder shows the contents of the image, and you can launch software from inside it. A `.pkg` file is an installer package, similar to a Windows `setup.exe`.

- `.dmg` is used to distribute software as a mounted disk image.
- `.pkg` launches an installation workflow when you double-click it.
- File extensions help you identify how macOS handles the file.

**Backup and maintenance tools**

macOS includes built-in tools that help with recovery and disk maintenance. Time Machine is the native backup utility. It keeps hourly backups for the last 24 hours, daily backups for the last month, and weekly backups after that. It continues until the backup drive fills, then removes the oldest backups as new ones are created. Disk Utility is used for disk maintenance when needed, and its First Aid function helps check and repair drive issues.

- **Time Machine** provides rolling backups.
- Hourly backups cover the last 24 hours.
- Daily backups cover the last month.
- Weekly backups continue after that until storage fills.
- **Disk Utility** is used as needed for drive maintenance.
- First Aid is the troubleshooting feature inside Disk Utility.

> Use Disk Utility when you have a drive problem rather than as a routine scheduled task.

**Support and compatibility considerations**

When you work with an iMac, verify the model and supported macOS version before planning upgrades or troubleshooting. Different Macs support different maximum macOS versions, and some systems may have limitations even when they can run a newer release. If you are unsure which Mac you have, identify the model first so you can confirm compatibility.

- Check the exact iMac model before choosing an OS version.
- Confirm the maximum supported macOS release for that system.
- Be cautious with versions that have limited or unsupported features.

**Quick review**

The iMac is best understood as Apple’s integrated desktop platform, with macOS tools and file formats that matter for support tasks. For A+ exam prep, focus on the relationship between the hardware form factor, `.dmg` and `.pkg` files, Time Machine, and Disk Utility.

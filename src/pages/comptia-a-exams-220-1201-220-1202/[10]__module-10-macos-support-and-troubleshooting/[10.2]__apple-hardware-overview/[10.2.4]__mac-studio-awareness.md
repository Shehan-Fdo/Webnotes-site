## **Course:** CompTIA A+ (Exams 220-1201 & 220-1202)

**Mac Studio in the Apple desktop lineup**

Mac Studio is a compact desktop workstation from Apple. It sits above the consumer-focused Mac mini and iMac, so you should recognize it as a higher-end Mac desktop rather than a general-purpose entry model.

- It is a **small-form-factor workstation**.
- It uses macOS as its operating system.
- It is part of Apple’s professional desktop family.

The key exam takeaway is placement: when you see Mac Studio, think “professional Mac desktop” and expect it to be used in more demanding workstation scenarios than a typical consumer Mac.

**macOS file types you should recognize**

macOS uses file extensions in the same general way as other operating systems. The extension helps identify what a file is used for.

- `.dmg` is an **Apple disk image**.
- A `.dmg` file can be mounted and appear as a drive in Finder.
- `.pkg` is an **installer package** used for software installation.
- `.zip` is a compressed archive format you may already know from other platforms.

> A `.dmg` is not just a compressed file; it behaves like a mountable disk image on macOS.

**Core macOS utilities and features**

macOS includes built-in tools that support usability, backup, and maintenance. These are common awareness items for A+.

- **Time Machine** provides rolling backups.
  - It keeps hourly backups for the last 24 hours.
  - It keeps daily backups for the last month.
  - It then keeps weekly backups as space allows.
  - When the backup drive fills up, the oldest data is removed as new data is added.
- **Disk Utility** is used for disk maintenance.
  - It is typically used as needed rather than on a fixed schedule.
  - Its First Aid function helps when you need to check or repair drive issues.

These tools matter because they show how macOS handles recovery and maintenance without relying on third-party utilities.

**Interface and productivity features**

macOS includes features that help you manage many open apps and separate tasks into different workspaces.

- **Mission Control** shows running applications in a spread-out view.
- You can open Mission Control by swiping up with three fingers or pressing <kbd>Control</kbd> + <kbd>↑</kbd>.
- **Spaces** lets you create multiple desktops.
- You can move apps between desktops from the Mission Control view.

This is useful when you want to keep work organized across separate desktops instead of crowding everything into one screen.

**Search, access, and account-related features**

Several macOS features support fast access to files and secure handling of credentials.

- **Spotlight** helps you search for content quickly.
- **Finder** is the main file-management interface.
- **Keychain** stores credentials and related secure information.
- **Continuity** helps Apple devices work together more smoothly.
- **iCloud** supports cloud-based integration across Apple devices.

For exam purposes, focus on what each feature does at a high level rather than memorizing deep configuration details.

**Practical recognition points**

When you are asked about macOS awareness, look for clues in the wording.

- A question about mounting an Apple disk image points to `.dmg`.
- A question about installing software from a package points to `.pkg`.
- A question about rolling backups points to Time Machine.
- A question about drive repair points to Disk Utility and First Aid.
- A question about viewing all open apps or separating desktops points to Mission Control and Spaces.

**Quick comparison**

| Item | What it is used for |
|---|---|
| `.dmg` | Mountable Apple disk image |
| `.pkg` | Installer package |
| Time Machine | Rolling backup system |
| Disk Utility | Disk maintenance and repair |
| Mission Control | View and manage open apps |
| Spaces | Multiple desktops |

**Study focus**

You do not need to treat Mac Studio as a separate operating system topic. Instead, connect it to the broader macOS environment and the tools, file types, and workflows that support Apple desktop administration.

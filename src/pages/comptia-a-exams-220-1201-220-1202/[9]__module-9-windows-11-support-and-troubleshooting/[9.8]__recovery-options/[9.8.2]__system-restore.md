## **Course:** CompTIA A+ (Exams 220-1201 & 220-1202)

**What System Restore does**

System Restore is a Windows recovery feature that returns the computer to a previous point in time. It is designed to help you recover from problems caused by recent changes, such as a driver update, software installation, or other configuration change.

It focuses on system-related data rather than user content. In practice, that means you use it to undo a bad change without wiping personal files.

- **System settings** are part of the rollback.
- **Installed drivers** can be reverted.
- **Registry** and configuration changes are included.
- Personal files such as documents, pictures, music, and videos are not the target of the restore.

**When to use it**

Use System Restore when Windows becomes unstable after a recent change and you want a practical rollback option before more disruptive recovery steps. It is especially useful when the issue appears after installing software or updating a driver.

- Recent driver changes can trigger instability.
- Recent application installs can introduce problems.
- Configuration changes can break normal behavior.
- It is a troubleshooting tool, not a full system replacement.

**How it fits into Windows recovery**

Microsoft lists System Restore alongside other recovery options in Windows Recovery Environment. That placement matters for exam prep because it shows System Restore is one option in a broader recovery toolkit, not the only one.

> System Restore does not replace backup. It helps you revert system state, but it is not a substitute for protecting important data.

**Key characteristics**

System Restore works on restore points, which are snapshots of system state taken at a previous time. Those snapshots are used to return Windows to a known-good configuration.

- Restore points represent a previous state.
- The feature is tied to system protection settings.
- In Windows 10 and Windows 11, it is turned off by default and must be enabled to function.
- It is available in desktop versions of Windows, but not as a universal recovery method for every Windows edition or scenario.

**Practical exam reminders**

For A+ troubleshooting, distinguish System Restore from a full recovery or reinstall. If the problem is tied to a recent system change and personal files are not the issue, System Restore is often the first recovery option to consider.

- Use it for rollback of system changes.
- Do not expect it to recover deleted personal files.
- Do not confuse it with full image recovery.
- Treat it as a fast way to return to a stable state.

**Related Windows access points**

Windows utilities can often be launched from the Run line or command line, and System Restore is part of that broader utility ecosystem. Knowing where recovery tools live helps you move faster during troubleshooting.

- Use the Windows interface to reach recovery tools.
- Use the Run line when you need quicker access.
- Use the command line when appropriate for utility launch and troubleshooting workflows.

**Summary**

System Restore is a configuration-focused recovery feature. It helps you undo recent system changes while leaving personal files alone, making it a useful first response to instability caused by drivers, software, or settings changes.

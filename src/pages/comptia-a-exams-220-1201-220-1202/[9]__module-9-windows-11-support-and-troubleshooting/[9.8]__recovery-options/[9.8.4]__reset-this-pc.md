## **Course:** CompTIA A+ (Exams 220-1201 & 220-1202)

**What “Reset this PC” does**

Windows includes a built-in reset feature that reinstalls the operating system to help fix software problems. It is commonly used when a system has persistent issues such as slow performance, malware infections, or other configuration-related problems.

A reset is different from a simple repair because it rebuilds Windows rather than only adjusting settings. In practice, you use it when troubleshooting steps have not resolved the issue and you need a more complete software recovery path.

- **Reinstall** Windows to restore the system to a usable state.
- Use it for issues tied to software, configuration, or corruption.
- Expect it to be more disruptive than basic troubleshooting tools.

**Reset options and outcomes**

Microsoft and OEM support documentation describe reset workflows that let you choose how much data to preserve. One common option is to keep personal files while reinstalling Windows. This helps you recover the operating system without removing user data, but it still replaces the Windows installation and can remove apps and settings.

> Always back up important data before you reset a PC. Even when personal files are preserved, the process can still affect installed applications and system configuration.

- **Keep personal files** preserves user data while reinstalling Windows.
- A reset can return the device to its original configuration.
- The process is useful when you want a clean software baseline without replacing hardware.

**Related recovery tools**

Reset this PC is part of a broader Windows recovery toolkit. Other recovery options include Startup Repair, System Restore, Recovery Drive, and reinstalling Windows with installation media. These tools solve different problems, so you should match the tool to the symptom.

System Restore is especially useful when a recent driver, update, or configuration change causes instability. It rolls back system settings, drivers, and registry-related changes without affecting personal files. That makes it less invasive than a full reset.

| Tool | Primary use | Personal files |
|---|---|---|
| Reset this PC | Reinstall Windows to fix broader software issues | May be kept, depending on option |
| System Restore | Roll back recent system changes | Not affected |
| Reinstall with installation media | Rebuild Windows from external media | Depends on installation method |

**When to choose reset versus restore**

Use System Restore first when the problem started after a recent change and you want to undo that change. Use Reset this PC when the system is still unstable, heavily misconfigured, or affected by persistent software problems that a restore point does not fix.

- Choose **System Restore** for recent driver or configuration issues.
- Choose **Reset this PC** for deeper software problems.
- Choose a full reinstall path when built-in recovery is not enough.

**Practical troubleshooting flow**

1. Back up important files.
2. Try less invasive recovery options first, such as System Restore.
3. If the issue remains, use Reset this PC.
4. Select the reset option that matches your data retention needs.
5. Verify that Windows starts correctly after the reset.

**Exam focus**

For A+ troubleshooting, remember the purpose of each recovery tool and the level of impact it has on the system. Reset this PC is a Windows recovery option that reinstalls the OS and is often used to resolve persistent software issues, while System Restore targets recent system changes.

- **Reset this PC** = reinstall Windows through built-in recovery.
- **System Restore** = revert system state without touching personal files.
- Backups matter before any major recovery action.

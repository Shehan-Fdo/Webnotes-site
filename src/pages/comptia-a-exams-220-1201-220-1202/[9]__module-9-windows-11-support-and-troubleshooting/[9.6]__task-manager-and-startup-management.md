## **Course:** CompTIA A+ (Exams 220-1201 & 220-1202)

**Task Manager overview**

Windows Task Manager is a real-time monitoring utility you use for troubleshooting and performance checks. It gives you quick visibility into what the operating system is doing right now, including CPU, memory, disk, network, processes, services, startup items, and users.

You use Task Manager when you need immediate feedback about system behavior. It helps you spot resource-heavy apps, confirm whether a service is running, and identify what may be slowing the computer down.

- **CPU** usage shows processor demand.
- **Memory** usage shows RAM consumption.
- **Disk** activity shows storage access.
- **Network** usage shows traffic activity.
- **Processes** lists running programs and background activity.
- **Services** shows Windows services.
- **Startup** shows apps that launch when Windows starts.
- **Users** shows logged-in user sessions.

**Opening Task Manager**

You can start Task Manager in several common ways. The fastest method is often the keyboard shortcut, but the other options are useful when you are already in Windows and need a mouse-based path.

1. Press <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>Delete</kbd> and select Task Manager.
2. Right-click the taskbar and choose Task Manager.
3. Press <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Esc</kbd> to open it directly.

> Use Task Manager as a first-pass troubleshooting tool when Windows feels slow, unresponsive, or resource constrained.

**What the tabs help you do**

Task Manager organizes system information into tabs so you can focus on one area at a time. This makes it easier to isolate whether the issue is tied to an app, a service, startup behavior, or overall system load.

- **Processes** helps you see what is actively consuming resources.
- **Performance** gives a broader view of system utilization.
- **Networking** helps you observe network activity.
- **Services** lets you review service status.
- **Startup** helps you manage what launches at boot.
- **Users** helps you see activity by user session.

The interface is designed for quick diagnosis. Instead of waiting for logs or deeper tools, you can check current conditions and decide whether a process, service, or startup item needs attention.

**Startup management**

Startup management focuses on the apps that launch automatically when Windows starts. Too many startup apps can increase boot time and make the system feel sluggish after sign-in because each app uses memory and CPU resources.

Managing startup items is a practical way to improve responsiveness. The goal is not to disable everything, but to reduce unnecessary background load while keeping essential software available.

- Startup apps can include security tools, convenience utilities, and background updaters.
- Each startup app consumes some RAM and CPU.
- Many startup apps can slow boot time and reduce responsiveness.
- Startup management can improve performance and support troubleshooting.

**Practical approach to startup items**

When you review startup apps, be conservative. Some items are necessary for security, hardware support, or vendor-specific features, while others are optional convenience tools.

1. Open Task Manager and review the **Startup** tab.
2. Identify apps that launch automatically.
3. Decide whether each item is required for normal operation.
4. Disable only items you do not need at startup.
5. Recheck system boot speed and responsiveness after changes.

**Troubleshooting value**

Task Manager is useful because it gives you second-to-second updates. If a system is slow, you can check whether CPU, memory, disk, or network usage is unusually high. If an app is crashing or the system is under load, Task Manager helps you narrow the problem before moving to deeper troubleshooting.

It is especially helpful when you need to compare normal behavior with current behavior. That makes it a strong first step for diagnosing performance issues in Windows.

**Key takeaways**

- Task Manager provides real-time system monitoring.
- You can open it with <kbd>Ctrl</kbd> + <kbd>Alt</kbd> + <kbd>Delete</kbd>, taskbar right-click, or <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Esc</kbd>.
- The **Startup** tab helps you control apps that launch with Windows.
- Fewer unnecessary startup apps can improve boot time and overall responsiveness.
- Use Task Manager early in Windows troubleshooting to identify resource bottlenecks.

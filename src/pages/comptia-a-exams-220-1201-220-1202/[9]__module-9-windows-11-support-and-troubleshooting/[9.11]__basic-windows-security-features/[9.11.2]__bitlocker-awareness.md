## **Course:** CompTIA A+ (Exams 220-1201 & 220-1202)

**BitLocker overview**

BitLocker is a Windows security feature that encrypts entire volumes. Its purpose is to reduce the risk of data theft or exposure when a device is lost, stolen, or improperly decommissioned.

- **Volume encryption** protects the data on a whole drive, not just individual files.
- It helps mitigate unauthorized access if someone removes the drive and connects it to another device.
- It is part of Windows security and is associated with protecting data at rest.
- BitLocker is relevant to Windows 10, Windows 11, and supported Windows Server versions.

> BitLocker is about protecting data on the device itself. It does not stop someone from using the computer if they already have authorized access.

**Why it matters**

If a laptop or desktop is lost, stolen, or retired without proper handling, the stored data can be exposed. BitLocker helps reduce that risk by making the contents of the volume unreadable without the proper protection in place.

- It addresses threats from physical loss of the device.
- It helps protect against offline attacks against the drive.
- It supports safer device decommissioning when hardware leaves service.

**What to remember for A+**

For exam awareness, focus on the role BitLocker plays rather than implementation details.

- **BitLocker** = Windows full-volume encryption.
- It is used to protect data on entire volumes.
- It is especially useful for portable devices and other systems that may be physically removed from a secure environment.
- It is one of the Windows enterprise-oriented security features you should recognize alongside other built-in protections.

**Practical recognition**

When you see a scenario involving a missing laptop, a stolen drive, or a device being sold or recycled, BitLocker is the feature that helps protect the data stored on that volume.

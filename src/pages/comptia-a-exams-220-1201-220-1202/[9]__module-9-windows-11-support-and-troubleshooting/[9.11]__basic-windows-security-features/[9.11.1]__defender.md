## **Course:** CompTIA A+ (Exams 220-1201 & 220-1202)

**Defender overview**

Microsoft Defender Antivirus is built into Windows 10 and Windows 11, so you already have baseline antivirus and anti-malware protection without buying a third-party product. In older references, you may still see the name Windows Defender, but the current name is Microsoft Defender Antivirus.

The key exam idea is that Defender should stay enabled and use **real-time analysis**. That means the antivirus checks files and applications as you download or launch them, helping identify malicious code before it runs.

- **Built-in** protection comes with Windows by default.
- **Real-time** scanning checks activity as it happens.
- **Virus definitions** must stay current for effective detection.
- The security app in Windows is where you manage this protection.

**What Defender does**

Defender provides protection against viruses and malware on the local device. Microsoft also positions Defender as part of a broader security ecosystem, including Microsoft Defender for Endpoint and other Defender services for organizations.

For A+ purposes, focus on the endpoint antivirus role first:
- It protects Windows devices without requiring separate antivirus software.
- It supports ongoing detection through updated definitions.
- It is part of the normal Windows security stack on Windows 10 and Windows 11.

**Managing Defender**

When troubleshooting, you may need to verify whether Defender is active, temporarily disabled, or updated. The practical management tasks emphasized in the evidence are activation, deactivation, and definition updates.

1. Open the Windows Security app.
2. Check that Microsoft Defender Antivirus is enabled.
3. Confirm real-time protection is on.
4. Update virus definitions if detection appears outdated.
5. Recheck protection after troubleshooting changes.

> If antivirus is disabled for testing or troubleshooting, restore protection as soon as possible.

**Defender in the Microsoft security family**

Microsoft uses the Defender name across several products and services. These are related but not identical:
- **Microsoft Defender Antivirus** protects Windows endpoints locally.
- **Microsoft Defender for Endpoint** is an enterprise endpoint security platform for preventing, detecting, investigating, and responding to advanced threats.
- **Microsoft Defender for Cloud** extends security coverage into cloud and code environments.
- **Microsoft Defender XDR** connects multiple Defender services into a broader security experience.

This broader naming can be confusing, so keep the scope clear. On the A+ exam, the most important distinction is that the built-in Windows antivirus is Microsoft Defender Antivirus.

**Exam focus**

Use these points to anchor your study:
- Defender is included with Windows 10 and Windows 11.
- It provides antivirus and anti-malware protection.
- It should remain enabled with real-time analysis.
- You may need to update virus definitions during troubleshooting.
- Older references may still say Windows Defender.

## **Course:** CompTIA A+ (Exams 220-1201 & 220-1202)

**What `chkdsk` does**

`chkdsk` is a Windows command-line utility that checks a volume’s file system and file system metadata for logical and physical errors. In exam terms, it is the command you use when you need to inspect disk integrity from the command prompt.

- It checks the file system and metadata on a volume.
- It can report status without making changes.
- It can fix errors when used with repair-related parameters.

> Interrupting `chkdsk` is not recommended. Microsoft notes that canceling or interrupting it should not leave the volume in a bad state, but you should still let it finish when possible.

**When it fixes errors**

If you run `chkdsk` without parameters, it only displays the volume status and does not repair anything. To make it fix errors, you use one of the repair-oriented switches.

- `/f` fixes file system errors.
- `/r` locates bad sectors and recovers readable information.
- `/x` forces the volume to dismount first.
- `/b` re-evaluates bad clusters on NTFS volumes.

A practical exam takeaway is that `chkdsk` is about disk and file system health, not partition management or memory diagnostics.

**Basic usage**

You run `chkdsk` from an elevated Command Prompt. Local Administrators membership, or equivalent, is the minimum requirement.

```cmd
chkdsk C:
chkdsk C: /f
chkdsk C: /r
```

- Use the plain command to view status.
- Add a repair switch when you want it to correct errors.
- Run it as administrator if you need the command to execute successfully.

**What to remember for A+**

`chkdsk` is commonly tested as the command for checking a disk for errors. It is associated with logical file errors and physical drive problems.

- `CHKDSK` checks disk errors.
- `DISKPART` manages partitions and disks, but it does not check for errors.
- `DEFRAG` reorganizes fragmented data.
- `MEM` reports memory usage.

**Quick comparison**

| Command | Main purpose |
|---|---|
| `chkdsk` | Checks a volume for file system and physical errors |
| `diskpart` | Manages disks and partitions |
| `defrag` | Defragments a drive |
| `mem` | Shows memory usage |

**Study tip**

If a question asks which command checks a hard drive for errors, choose `chkdsk`. If the question asks whether it repairs problems, look for a switch such as `/f` or `/r`.

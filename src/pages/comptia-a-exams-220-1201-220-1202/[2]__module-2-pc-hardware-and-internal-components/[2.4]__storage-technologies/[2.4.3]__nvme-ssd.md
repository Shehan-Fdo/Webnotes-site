## **Course:** CompTIA A+ (Exams 220-1201 & 220-1202)

**NVMe overview**

<abbr title="Non-Volatile Memory Express">NVMe</abbr> is a modern storage protocol designed specifically for <abbr title="Solid-State Drive">SSD</abbr> use. It takes advantage of the fact that SSDs do not have the mechanical limits of hard disk drives, so it can communicate more efficiently with the system.

- NVMe uses a streamlined command set.
- It supports more parallel operations than older storage approaches.
- It can deliver faster read and write speeds than SATA-based SSDs.
- It is commonly associated with M.2 or U.2 connections that use <abbr title="Peripheral Component Interconnect Express">PCIe</abbr> lanes for performance.

**How NVMe differs from other storage**

NVMe is about the protocol and communication path, not just the physical drive shape. That distinction matters when you compare it with SATA-based SSDs or when you troubleshoot a drive that is not detected.

| Topic | NVMe SSD | SATA-based SSD |
|---|---|---|
| Communication | Uses a streamlined NVMe protocol | Uses SATA communication |
| Parallelism | Supports more parallel operations | Less optimized for parallel access |
| Performance | Designed for higher speed | Typically slower than NVMe |
| Typical connection | M.2 or U.2 with PCIe lanes | SATA data and power connections |

A drive can be physically small and still not be NVMe. When you identify storage hardware, check both the interface and the protocol.

**Detection and troubleshooting**

If the system cannot find an M.2 SSD after power-on, start with firmware and configuration checks before assuming the drive has failed. Vendor guidance emphasizes BIOS-level troubleshooting and updating the latest BIOS when needed.

1. Confirm the drive is installed correctly.
2. Check BIOS or <abbr title="Basic Input/Output System">BIOS</abbr> settings for storage detection.
3. Update the motherboard BIOS if the vendor recommends it.
4. Verify the operating system can see the drive in tools such as Disk Management.
5. If the drive still does not appear, test with another known-good drive or enclosure when possible.

> If an external M.2 NVMe drive stops appearing in Windows, the issue may involve the enclosure, controller, or system configuration rather than the SSD itself.

**Exam focus**

For A+ purposes, focus on the practical relationship between NVMe, SSD performance, and motherboard support.

- NVMe is a storage protocol for SSDs.
- It is faster than SATA-based SSDs because it is built for efficient communication and parallel operations.
- M.2 is a common form factor or connection style, but it does not automatically mean NVMe.
- Troubleshooting often begins in BIOS when a drive is not detected after startup.

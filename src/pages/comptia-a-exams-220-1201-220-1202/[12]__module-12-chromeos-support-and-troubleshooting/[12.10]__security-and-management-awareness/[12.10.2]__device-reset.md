## **Course:** CompTIA A+ (Exams 220-1201 & 220-1202)

**Device reset overview**

A device reset on a Chromebook is a factory reset, also called a **Powerwash**. It returns the device to a fresh state and can make it run like new again. This is useful when you want to remove user data, fix persistent profile or settings problems, or prepare the device for a new owner.

A reset is not the first troubleshooting step. If the Chromebook still works normally after a restart, or if the issue may be caused by a browser extension, try simpler fixes first.

- Use a reset when you see the message `Reset this Chrome device`.
- Consider it when user profile or settings problems persist.
- Use it before transferring ownership of the Chromebook.
- If the device is managed by a work or school, you cannot reset it yourself.

**What a factory reset removes**

A factory reset erases all user data on the Chromebook's hard drive, including files in the `Downloads` folder. It also removes most non-user-related device information, while preserving a few diagnostic and operational pieces of data.

It does not delete files stored in Google Drive or on an external storage device. That makes backup planning important before you start the reset.

> Important: If the Chromebook is used at work or school, contact the administrator to wipe the device and re-enroll it on the network.

**Before you reset**

Back up anything you want to keep before you begin. The safest options are Google Drive or an external hard drive. Since the reset removes local files, you should confirm that important data is stored somewhere else first.

You should also try other troubleshooting steps before committing to a reset. For example, disabling Chrome extensions one at a time can help identify whether an extension is causing the problem.

- Back up local files from `Downloads`.
- Keep files in Google Drive or external storage if you want them preserved.
- Test whether a browser extension is the source of the issue.
- Use the reset only after simpler recovery steps fail.

**Practical reset decision points**

A reset is most appropriate when the device needs a clean start rather than a targeted repair. That includes cases where the Chromebook is being reassigned, the user profile is corrupted, or normal operation does not return after basic troubleshooting.

<p>
If the problem is limited to a browser extension or a temporary glitch, start with a smaller fix. If the device needs to be handed off, re-enrolled, or fully cleared of user data, a factory reset is the correct choice.

**Key takeaways**

- A Chromebook factory reset is the same as a Powerwash.
- It erases local user data, including `Downloads`.
- It preserves files in Google Drive and on external storage.
- Managed work or school Chromebooks require administrator action.
- Back up first, then use the reset when simpler fixes do not solve the issue.

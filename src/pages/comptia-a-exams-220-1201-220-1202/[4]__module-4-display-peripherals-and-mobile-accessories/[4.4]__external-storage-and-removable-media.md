## **Course:** CompTIA A+ (Exams 220-1201 & 220-1202)

**External storage overview**

External storage devices connect to computers and mobile devices through a range of interfaces. In A+ contexts, you should recognize that these devices are often portable, easy to replace, and used for moving data between systems. Removable storage has been part of personal computing for a long time, from floppy disks to modern USB-based drives and card readers.

- **USB** is one of the most common connection types for external storage.
- **FireWire** and **eSATA** also appear in removable drive discussions.
- **Flash memory** is common in small portable storage devices and can be overwritten easily, so backups matter.

**Connection types and device interfaces**

USB, or <abbr title="Universal Serial Bus">USB</abbr>, has gone through multiple iterations and uses different connector shapes. Some connectors are very small, such as micro USB, which is common on mobile devices because of its compact size. You also need to watch for proprietary connectors that may resemble USB but are not standard USB connections.

> Be careful not to assume that a small connector is USB just because it looks similar. Verify the device and cable type before connecting.

Common external storage interfaces include:

- **USB** for broad compatibility across computers and mobile devices
- **micro USB** for compact device connections
- **FireWire** for legacy portable storage use
- **eSATA** for external storage with high throughput

**Removable media types**

Removable storage includes media you can insert and remove from a host device. Older examples include floppy disks, while modern examples include optical media and flash-based cards. The key exam idea is that removable media is not fixed inside the system and is intended for portability.

- **Floppy disk drives** were widely used in earlier personal computers.
- **CDs** and **DVDs** are removable optical media.
- **Memory cards** are removable flash media used in cameras, phones, and other devices.

Floppy disks are a historical example of removable storage. They used magnetic media inside a protective shell and came in different sizes and capacities over time. While they are no longer common in modern systems, they help illustrate the difference between fixed and removable storage.

**Memory card readers and adapters**

Memory card readers provide a bridge between removable card formats and host devices. They support workflows where you need to transfer data quickly between a card and a computer. These readers are common accessories for SD and microSD media.

- **Card readers** move data between compatible memory card formats and host devices.
- **SD** and **microSD** are common card formats supported by readers.
- **USB-C** and USB 3.0 card readers are common on modern systems.

A card reader may support multiple formats, such as SD, microSD, SDHC, SDXC, and MMC. The important practical point is compatibility: the reader, card type, and host port must all work together.

**Practical exam focus**

When you identify external storage on the exam, think in terms of connection type, media type, and compatibility. A device may use a standard USB connection, a smaller mobile connector, or a dedicated card reader. The safest approach is to match the connector and media to the device rather than assuming all small or removable storage uses the same interface.

1. Identify the storage medium: hard drive, flash drive, memory card, optical disc, or legacy floppy.
2. Check the connection type: USB, micro USB, FireWire, eSATA, or card reader interface.
3. Confirm compatibility between the host device, cable, and removable media format.
4. Remember that flash-based removable media can be overwritten, so protect important data with backups.

**Quick comparison**

| Item | What it is | Exam takeaway |
|---|---|---|
| USB external drive | Portable storage using USB | Most common external connection |
| micro USB device | Small connector for compact devices | Common on mobile devices |
| Card reader | Adapter for memory cards | Transfers data between card and host |
| Optical media | CD or DVD | Removable, disc-based storage |
| Floppy disk | Legacy removable media | Historical example of removable storage |

**Key reminders**

- **Compatibility** matters more than appearance when connectors are small or proprietary.
- **Removable media** is portable and not permanently installed in the system.
- **Backup planning** is important because flash memory can be overwritten easily.
- **Throughput** can vary by interface, with eSATA noted for fast external storage performance.

## **Course:** CompTIA A+ (Exams 220-1201 & 220-1202) — V15

**Cloud storage and synchronization overview**

Cloud storage lets you keep files in a remote service and access them from multiple devices. Synchronization keeps the local copy and cloud copy aligned so changes made in one place appear in the other after the device reconnects. For A+ troubleshooting, focus on how sync clients behave, what is available offline, and what happens when sync stops working.

- **OneDrive** is Microsoft’s sync service for personal and work files.
- **SharePoint** and Teams files can also sync to a computer through OneDrive for work or school.
- Sync lets you work in File Explorer and then update files automatically when you go back online.
- Problems often show up as missing files, delayed updates, or files split between local and cloud locations.

**OneDrive sync basics**

OneDrive can sync files and folders between your device and the cloud. Microsoft also provides options to back up folders, choose which folders to sync, pause syncing, stop syncing, and change the OneDrive folder location. These controls matter when you need to reduce local storage use or isolate a sync problem.

- You can upload files and folders to OneDrive and download them back to a device.
- You can choose which folders sync to avoid pulling down everything.
- You can pause syncing when you need to stop changes temporarily.
- You can reinstall OneDrive if the client itself is damaged or misbehaving.

**SharePoint and Teams synchronization**

If your organization uses Microsoft 365, you can sync SharePoint and Teams files to a folder on your computer. This gives you direct access in File Explorer and supports offline work. When you reconnect, changes sync automatically.

> Support for the old OneDrive for Business sync app with SharePoint Online has ended.

- Sync is typically done at the library level rather than by syncing an entire site at once.
- Each synced library appears as a folder on the local computer.
- If you need a cleaner structure, plan the library layout before syncing.

**Files On-Demand**

<abbr title="Files On-Demand">Files On-Demand</abbr> lets you see all your SharePoint files without downloading every file to the device. This saves local storage and keeps files available as online-only until you open them.

- Online-only files do not take up space on your computer.
- When you open an online-only file, it downloads to the device.
- Status icons show which files are local, online-only, or syncing.
- You can copy or move files to SharePoint directly from the file system.

**Common sync management tasks**

Use the OneDrive client settings to control how sync behaves on a Windows device. The basic workflow is to open the OneDrive cloud icon in the notification area, then adjust sync-related settings as needed.

1. Select the OneDrive cloud icon in the Windows notification area.
2. Open `Settings`.
3. Choose the sync or storage option you need, such as saving space by downloading files as you use them.
4. Verify that the expected folders and libraries are selected for sync.
5. Check whether syncing is paused or stopped if changes are not appearing.

**Troubleshooting sync problems**

When files do not sync correctly, start with the simplest checks first. Confirm that the device is connected, the correct account is signed in, and the sync client is running. If files appear split between local and cloud copies, the issue may be related to sync state, folder selection, or a client problem.

- Check for sync status icons and error indicators.
- Confirm that the right folders or libraries are selected.
- Look for paused syncing or a stopped sync relationship.
- If the client seems stuck, reinstalling OneDrive may help.
- If the issue affects SharePoint or Teams files, verify that the organization uses a supported sync method.

**Exam focus**

For CompTIA A+, know the difference between storage and synchronization. Cloud storage is where the file lives; synchronization is the process that keeps copies aligned across devices. Also know that Files On-Demand reduces local storage use, and that OneDrive is the common Microsoft tool for syncing personal, SharePoint, and Teams content.

- **Cloud storage** provides remote file access.
- **Synchronization** keeps copies consistent across devices.
- **Files On-Demand** saves space by keeping files online-only until needed.
- **OneDrive** is the main Microsoft sync client you should recognize for exam scenarios.

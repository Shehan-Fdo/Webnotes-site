## **Course:** Comptia Network Plus

**Cabling media overview**

- Network cables and connectors let devices connect and communicate.
- Common media include **copper** and fiber.
- Cable choice affects standards, limitations, and speed.
- Network cabling also supports data transfer between computers, routers, switches, and storage area networks.

**Twisted pair cabling**

- Twisted pair uses two wires twisted together.
- Twisting helps cancel electromagnetic interference (`EMI`) and crosstalk.
- It is common in Ethernet networks and telephone systems.
- Twisted pair comes in `UTP` (Unshielded Twisted Pair) and `STP` (Shielded Twisted Pair).
- `UTP` is the most common type.
- `STP` adds an outer metal foil or braided shield for more EMI protection.
- Twisted pair cables use color coding to help installation and maintenance.
- The `TIA/EIA-568` standard defines twisted pair cabling performance.
- Common category examples include `Cat5e`, `Cat6`, and `Cat6a`.

**Ethernet cable categories**

- Ethernet cables use numbered categories, or `cat`, to show specifications.
- Higher category numbers generally indicate higher performance.
- `Cat3` is obsolete for modern high-speed networks.
- `Cat3` is associated with older `10BaseT` networks.
- `Cat5` supports up to `100 Mbps` over `100 meters`.
- `Cat5e` supports up to `1 Gbps` over `100 meters`.
- `Cat6` supports `10/100/1000 Mbps` and can support `10 Gbps` over `100 meters`.
- Category labels help you choose the right cable for the application.

**Fiber optic cabling**

- Fiber optic cabling uses light pulses instead of electrical signals.
- Devices convert digital signals to light for transmission and convert them back at the other end.
- Fiber is difficult to tap without detection.
- A signal drop can help reveal unauthorized access.
- Fiber is a strong choice when you need high throughput and long-distance transmission.
- Fiber connectors include several common types.
- `ST` stands for straight tip.
- `ST` connectors use a bayonet-style lock.
- You insert and twist `ST` connectors to secure them.

**Connector and cable handling**

- Different network media use different connector types.
- Copper and fiber networks both use many connection styles.
- Cable management can include patch panels and punchdown blocks.
- Connector selection depends on the cable type and the network design.
- Media converters can bridge different cabling types when needed.

**Practical study points**

- Match the cable type to the required speed and distance.
- Check whether the environment needs extra EMI protection.
- Use category ratings to compare Ethernet cable capability.
- Use fiber when you need better resistance to tapping and strong long-distance performance.
- Use connector types that fit the media and lock securely in place.

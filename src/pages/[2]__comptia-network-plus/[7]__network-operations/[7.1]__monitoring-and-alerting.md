## **Course:** Comptia Network Plus

**Monitoring goals**

- Monitor networks for **high availability** and performance issues.
- Check whether devices stay up and running.
- Watch for load, utilization, and security concerns.
- Use monitoring to support troubleshooting and maintenance.

**Common monitoring data**

- Collect **flow data** to summarize traffic patterns.
- Use protocol analysis to inspect network activity.
- Review log file information from devices.
- Query devices for status and operational details.

**Monitoring tools and sources**

- Use **SNMP** (Simple Network Management Protocol) to query device information.
- Use **NetFlow** to gather summarized traffic data.
- Use **Syslog** to centralize log messages from network devices.
- Use a **SIEM** (Security Information and Event Management) platform to roll up and analyze events.
- Use a central console or network management station to view multiple data sources in one place.

**What you can learn from monitoring**

- Confirm whether users can authenticate properly.
- Identify possible security concerns.
- Measure CPU or network utilization.
- Check whether a device responds on the network.
- Determine whether ports are open or closed.

**Practical monitoring approach**

1. Identify the device or service you need to observe.
2. Collect status, log, or flow information from the source.
3. Review the data in a central console.
4. Look for availability issues, abnormal load, or security signals.
5. Use the findings to guide troubleshooting.

**Alerting focus**

- Alerting helps you notice problems when devices or services fail.
- Use alerts for conditions that affect availability or security.
- Keep alerts tied to meaningful thresholds or events.
- Avoid relying on a single data source when you need a fuller picture.

**Exam focus**

- Network+ emphasizes monitoring network activity and troubleshooting performance and availability issues.
- Expect questions about tools that collect device status, traffic summaries, and logs.
- Know the difference between raw logs, summarized flow data, and centralized event analysis.

## **Course:** Comptia Network Plus

**IPv4 fundamentals**

- IPv4 stands for Internet Protocol version 4.
- An IPv4 address is a **logical address** used for routing packets across IP networks.
- IPv4 addresses use dotted decimal notation.
- A standard IPv4 address has 32 bits.
- Those 32 bits are split into four 8-bit blocks.
- Each 8-bit block is an **octet**.
- Each octet can range from `0` to `255`.
- No octet can exceed `255` because it contains 8 bits.
- IPv4 addresses help identify both the network and the host.

**Address structure**

- The network portion identifies the network segment.
- The host portion identifies the specific device on that network.
- A host is a device with a unique IP address on the network.
- Binary representation underlies the decimal format you see in IPv4 addresses.
- You can use binary math to calculate subnet values.
- Subnet calculations include the network address, broadcast address, and device addresses.

**Address classes**

- Class A uses the leading bit pattern `0`.
- Class B uses the leading bit pattern `10`.
- Class C uses the leading bit pattern `110`.
- Class A addresses range from `1.0.0.1` to `126.255.255.254`.
- Class B addresses range from `128.1.0.1` to `191.255.255.254`.
- Class C addresses range from `192.0.1.1` to `223.255.254.254`.
- Class D is listed as `N.A.` in the provided evidence.
- Class A supports the largest host count per network among the listed classes.
- Class C supports the smallest host count per network among the listed classes.

**Practical exam focus**

- Expect to recognize the 32-bit, four-octet IPv4 format.
- Expect to identify valid octet values from `0` to `255`.
- Expect to distinguish network and host portions in an address.
- Expect to relate binary math to subnetting tasks.
- Expect to recognize classful address ranges when reviewing IPv4 basics.

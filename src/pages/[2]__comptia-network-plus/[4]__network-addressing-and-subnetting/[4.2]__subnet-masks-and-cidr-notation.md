## **Course:** Comptia Network Plus

**Subnet mask basics**

- A subnet mask uses contiguous `1` bits on the left and `0` bits on the right.
- An IPv4 address is `32` bits long.
- Each IPv4 octet is `8` bits.
- Each octet ranges from `0` to `255`.
- A mask like `255.255.255.0` uses three full octets of `1` bits.

**CIDR notation**

- **CIDR** means Classless Inter-Domain Routing.
- CIDR notation writes the number of mask bits after a slash.
- `255.255.255.0` is the same as `/24`.
- `/24` means `24` bits are part of the subnet mask.
- You may also see CIDR called prefix notation or slash notation.
- `255.0.0.0` is the same as `/8`.

**Classless subnetting**

- Since `1993`, networks use classless subnetting instead of class-based subnetting.
- Classless subnetting gives more efficiency and flexibility with IP addressing.
- You can assign masks that do not fit only class A, class B, or class C patterns.
- Variable Length Subnet Masks (**VLSM**) let you segment a network into smaller IP subnets.

**Why subnetting matters**

- You use subnetting to create smaller networks.
- Routers move traffic between those smaller networks.
- Subnetting helps you avoid needing one device to know every destination in the world.
- Smaller subnets make IP addressing easier to manage.

**Common notation examples**

- `192.168.1.44/24` combines an IPv4 address with CIDR notation.
- The `/24` part tells you how many bits belong to the subnet mask.
- A CIDR value can range from `0` to `32` bits for IPv4.
- A mask can also be written in dotted-decimal form or CIDR form.

**Practical conversion focus**

- You should be able to translate between dotted-decimal notation and CIDR-block notation.
- You should recognize that the same subnet mask can appear in more than one format.
- You should use binary understanding to move between mask representations.

**Key takeaways**

- A subnet mask marks the network portion of an IPv4 address.
- CIDR notation shortens the mask to a slash and bit count.
- Classless subnetting supports modern IP allocation and routing.
- VLSM helps you build smaller subnets with more flexibility.

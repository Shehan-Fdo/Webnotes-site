## **Course:** Comptia Network Plus

**Virtualization and network change**

- Virtualization lets you replace many physical servers with fewer physical devices running multiple **virtual servers**.
- Network design changes when servers become virtualized.
- Modern networks often use multiple VLANs, redundant connections, and high-speed communication.
- Virtualization is closely related to cloud computing and other IT infrastructure topics.

**Core virtual networking concepts**

- **Network function virtualization (NFV)** moves network functions into virtualized software.
- A **hypervisor** creates and manages virtual machines on a host system.
- A **vSwitch** connects virtual machines within the virtual environment.
- A **vNIC** is a virtual network interface card used by a virtual machine.
- These components help virtual systems communicate without relying on only physical network hardware.

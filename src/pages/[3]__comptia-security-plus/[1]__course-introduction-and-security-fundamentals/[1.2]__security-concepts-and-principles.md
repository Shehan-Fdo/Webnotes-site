## **Course:** CompTia Security Plus

**Security fundamentals**

- Security+ focuses on practical, hands-on skills for real-world security tasks.
- The exam emphasizes securing networks, applications, and devices.
- You use security concepts to protect enterprise environments.
- Core knowledge includes threats, vulnerabilities, and controls.

**CIA triad**

- The **CIA triad** stands for confidentiality, integrity, and availability.
- **Confidentiality** limits access to authorized users.
- **Integrity** protects data from unauthorized change.
- **Availability** keeps systems and data accessible when needed.

**Access control concepts**

- Identity and access management is a core Security+ topic.
- **Authentication** verifies identity.
- **Authorization** defines what an identity can do.
- **Accounting** tracks actions and access activity.
- Multi-factor authentication adds more than one verification factor.

**Threats and vulnerabilities**

- Security+ covers malware, social engineering, and application attacks.
- You identify threats before you can reduce risk.
- Vulnerabilities create exposure that attackers can exploit.
- Security tools help you detect and limit attack impact.

**Security technologies and tools**

- Firewalls help control network traffic.
- Intrusion detection systems help identify suspicious activity.
- Endpoint security helps protect devices.
- Security tools support prevention, detection, and response.

**Secure architecture and protocols**

- You design secure network architectures to reduce exposure.
- You implement secure systems to support protection goals.
- You apply secure protocols where appropriate.
- Architecture choices should support confidentiality, integrity, and availability.

**Risk and continuity**

- Risk analysis helps you assess likely security impact.
- Mitigation strategies reduce risk to acceptable levels.
- Business continuity planning supports operations during disruption.
- Security decisions should balance protection and practicality.

**Security mindset**

- Build a broad foundation before narrowing into a specialty.
- Focus on concepts that apply across roles and environments.
- Use security skills to support daily IT work and long-term career growth.

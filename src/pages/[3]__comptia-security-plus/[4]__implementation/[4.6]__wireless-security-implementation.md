## **Course:** CompTia Security Plus

**Wireless security goals**

- Wireless traffic travels over the air, so nearby attackers may listen in.
- Encrypt traffic by default on private wireless networks.
- Use protocols that protect **integrity** as well as confidentiality.
- Require authentication before granting network access.

**Wireless encryption and protocol selection**

- Use modern wireless security protocols instead of outdated ones.
- WEP is weak because it reuses keys and is vulnerable to capture-based attacks.
- WPA adds **TKIP** (Temporal Key Integrity Protocol) to improve key handling.
- WPA3 is the latest wireless security standard in the evidence set.
- Avoid outdated protocols when stronger options are available.

**Authentication framework**

- **EAP** (Extensible Authentication Protocol) is a framework for wireless authentication.
- EAP supports multiple authentication types.
- WPA and WPA2 can use EAP-based authentication methods.
- **802.1X** provides port-based network access control.
- **RADIUS** is commonly used as the authentication server in enterprise wireless setups.

**Common access methods**

- **PSK** (Pre-Shared Key) uses the same key on the access point and client.
- PSK is simple to deploy.
- 802.1X/EAP provides stronger control than PSK in enterprise environments.
- MAC address filtering can allow or block devices by MAC address.
- MAC address filtering is limited as a sole security control.

**Wireless threats and exposure**

- Rogue access points can be added without authorization.
- An evil twin can imitate a legitimate wireless network.
- Unauthorized wireless devices can create a path into the corporate network.
- Wireless networks are hard to physically secure because radio signals extend beyond walls.

**Practical implementation priorities**

1. Encrypt wireless traffic.
2. Require strong authentication.
3. Prefer enterprise controls such as `802.1X` and `RADIUS` when appropriate.
4. Avoid relying on weak or outdated wireless standards.
5. Watch for unauthorized access points and devices.

**Exam focus**

- Know the purpose of **EAP**, `802.1X`, and `RADIUS`.
- Know the difference between `PSK` and enterprise authentication.
- Know why WEP is insecure and why WPA3 matters.
- Know that rogue access points and evil twins are wireless security concerns.

## **Course:** CompTia Security Plus

**Endpoint security overview**

- Endpoint security protects devices such as desktops, laptops, tablets, and mobile phones.
- It helps defend against inbound attacks and outbound data theft.
- It uses **layered protection** to reduce risk across different platforms and operating systems.
- Security controls vary by device type and operating system.

**Common endpoint protections**

- **Anti-malware** and antivirus tools help detect and block malicious software.
- **EDR** (Endpoint Detection and Response) supports detection and response on endpoints.
- **Data loss prevention** helps stop sensitive information from leaving the device.
- **Host-based firewalls** filter traffic on the local device.
- Additional endpoint controls may be added as part of a defense-in-depth strategy.

**Hardening and attack surface reduction**

- Endpoint hardening makes devices harder to exploit.
- You should apply security updates and patches regularly.
- You should secure user accounts on endpoint systems.
- Application hardening reduces the number of attack surfaces.
- Hardening can include port filters, registry settings, disk encryption, and patch management.
- Compliance requirements such as HIPAA or PCI-DSS can guide hardening choices.

**Practical security focus**

- Endpoint security supports confidentiality, integrity, and availability.
- It is part of the tools used to protect systems in Security+.
- Practical controls matter more than theory in real environments.
- You should choose controls that match the device, platform, and risk.

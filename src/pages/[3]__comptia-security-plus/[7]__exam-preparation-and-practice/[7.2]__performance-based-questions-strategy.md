## **Course:** CompTia Security Plus

**Performance-Based Questions overview**

- PBQs are exam items that test your ability to solve problems in real-world settings.
- They appear as **simulations** or within **virtual environments**.
- Simulations approximate tools or systems such as a firewall, network diagram, terminal window, or operating system.
- Simulations usually have restricted functionality.
- Simulations can allow multiple valid responses or paths.
- Virtual environments use full versions of the technology assessed by the item.
- Virtual environments can allow incorrect steps or paths.
- CompTIA uses simulation PBQs in Security+ and other certifications.
- CompTIA also uses virtual and simulation PBQs in some certifications.

**What to expect on exam day**

- PBQs feel different from multiple-choice questions.
- PBQs ask you to do something, not just recognize an answer.
- PBQs can feel unexpected if you only practice flashcards.
- PBQs reward applied understanding over memorization.
- PBQ readiness can affect your pace early in the exam.
- You should expect to think through the task before acting.

**Strategy for handling PBQs**

- Read the prompt carefully before you interact with the item.
- Identify the goal of the scenario first.
- Look for the tool, interface, or environment type.
- Use a methodical approach instead of guessing.
- Try the most likely path when the item allows multiple responses.
- Avoid wasting time on unfamiliar controls.
- Move on if a question stalls your progress.

**Preparation approach**

- Practice with timed tests to build exam readiness.
- Use PBQ-style practice to reduce surprise on exam day.
- Review weak areas after practice.
- Focus on understanding how security concepts apply in scenarios.
- Build confidence by working through interactive tasks, not only multiple-choice items.

**Study reminders**

- PBQs test hands-on skill in realistic situations.
- CompTIA does not publish actual PBQs.
- PBQ forms vary by exam version and candidate.
- Treat sample PBQs as practice for the experience, not as real exam content.
- Use adaptive practice to target knowledge gaps.

**Key terms**

- **PBQ**: Performance-Based Question.
- **Simulation**: A restricted environment that approximates a tool or system.
- **Virtual environment**: A full version of the assessed technology running in a production-like setting.

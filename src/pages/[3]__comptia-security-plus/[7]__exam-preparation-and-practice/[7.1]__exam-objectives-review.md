## **Course:** CompTia Security Plus

**Exam focus**

- Security+ validates baseline skills for core security functions.
- The exam emphasizes practical, hands-on skills.
- It targets real-world security challenges.
- It supports an IT security career path.
- It is widely recognized in cybersecurity.

**Core skill areas**

- Identify threats, attacks, and vulnerabilities.
- Include **malware** and social engineering.
- Recognize application attacks.
- Use security technologies and tools.
- Include firewalls, intrusion detection systems, and endpoint security.
- Design secure network architectures.
- Implement secure systems and secure protocols.
- Manage identity and access concepts.
- Cover authentication, authorization, and accounting.
- Assess and manage risk.
- Use risk analysis, mitigation, and business continuity planning.
- Apply cryptography and public key infrastructure.

**What the exam expects**

- Detect compromise and understand vulnerability scanning.
- Understand penetration testing concepts.
- Install and configure network components.
- Troubleshoot issues that affect organizational security.
- Install identity and access services.
- Configure management controls.
- Summarize risk management best practices.
- Explain business impact.
- Configure wireless security settings.

**Study approach**

- Review the objectives as connected domains, not isolated topics.
- Cross-check topics against the current Security+ objectives.
- Use sample questions to see how objectives appear on the exam.
- Focus on practical application, not memorization alone.
- Prioritize areas that recur across security roles.

**Key terms**

- **AAA** means authentication, authorization, and accounting.
- **PKI** means public key infrastructure.
- **DoD** means U.S. Department of Defense.
- **Baseline skills** means the minimum core skills expected for security work.

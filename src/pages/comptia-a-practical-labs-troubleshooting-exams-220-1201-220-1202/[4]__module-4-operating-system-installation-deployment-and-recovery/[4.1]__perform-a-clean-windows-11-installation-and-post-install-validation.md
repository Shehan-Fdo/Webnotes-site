## **Course:** CompTIA A+ Practical Labs & Troubleshooting (Exams 220-1201 & 220-1202)

**Clean installation workflow**

A clean Windows 11 installation replaces the existing OS setup on a device and is commonly used during re-imaging or when you want a fresh start. Before you begin, confirm that the device is intended for Windows 11 and that you have a bootable installation USB ready.

Microsoft notes that Windows 11 has minimum system requirements, and devices that do not meet them may not install successfully. If you are unsure, check compatibility before you wipe the machine.

- Verify the device meets Windows 11 requirements before installation.
- Use installation media that boots correctly into Windows Setup.
- Back up important data before re-imaging or reinstalling.
- Keep recovery options in mind if the install does not proceed normally.

**Pre-install checks**

A practical clean-install process starts with validation, not with the installer. Microsoft recommends checking device specifications and compatibility, and support guidance also points to using the PC Health Check app when the device is already running Windows 10.

If you are reinstalling a system for a user, back up data first. A re-image or wipe removes local files unless you preserve them separately.

- Confirm hardware compatibility with Windows 11 requirements.
- Check whether the device is already on Windows 10 version 2004 or later if you plan to upgrade instead of clean install.
- Save browser data, documents, and other important files before wiping the drive.
- Make sure you have access to the correct account credentials after reinstall.

**Boot media and setup behavior**

When you boot from the Windows 11 USB installer, Setup should normally move into the installation flow and present the target disk selection. If Setup immediately asks for a driver, treat that as a sign to check the storage path, BIOS settings, and the installation media.

In one reported case, the SSD was visible in BIOS and `diskpart`, but Windows Setup still requested a driver instead of showing the install location. That kind of mismatch suggests a storage or setup-environment issue rather than a missing drive in the system itself.

> If Windows Setup asks for a driver unexpectedly, do not assume the SSD is absent. Check firmware settings, boot mode, and the installer media first.

- Confirm the USB installer is the active boot device.
- Review BIOS or UEFI settings if Setup does not reach the disk selection screen.
- Check whether secure boot or other firmware settings affect the boot path.
- Use `diskpart` only as a diagnostic tool when Setup behavior is abnormal.

**Installation validation**

After Windows 11 installs, validate that the system is usable before handing it back to the user. Post-install validation is about confirming the machine boots normally, storage is accessible, and the system is ready for updates and user sign-in.

A clean install is not complete until the device behaves normally in the new environment. If the machine was rebuilt after hardware changes, re-check compatibility and confirm the system still meets Windows 11 requirements.

- Boot into the installed Windows environment successfully.
- Confirm the internal drive is visible and usable.
- Verify the system can sign in with the expected account.
- Check that the device still meets Windows 11 requirements after hardware changes.

**Troubleshooting and recovery options**

If the installation or first boot fails, use the least disruptive recovery path available. Microsoft community guidance points to Safe Mode with Networking when possible, and to WinRE (Windows Recovery Environment) or bootable media when normal startup is not available.

Safe Mode can help when you need internet access for troubleshooting, while WinRE provides repair tools such as Command Prompt, System Restore, and uninstall options.

1. Try Safe Mode with Networking if the system can still boot far enough to use it.
2. Use Safe Mode if networking is not required.
3. If normal boot fails, open WinRE and use repair tools.
4. Fall back to bootable media when the installed OS cannot be repaired from inside Windows.

**Post-install checklist**

After the clean install, validate the basics in a repeatable order. This helps you catch missing drivers, account issues, or storage problems before the device returns to production use.

- Confirm Windows starts without setup prompts.
- Check that the expected storage devices appear in the system.
- Verify user data restoration if the device was re-imaged.
- Apply updates and confirm the system remains stable after reboot.

**Key takeaways**

A clean Windows 11 installation depends on preparation, compatibility checks, and careful validation after setup. If Setup behaves unexpectedly, focus on firmware, boot media, and recovery tools before assuming the hardware has failed.

The safest workflow is to back up data, confirm requirements, boot from known-good media, and verify the system after installation.

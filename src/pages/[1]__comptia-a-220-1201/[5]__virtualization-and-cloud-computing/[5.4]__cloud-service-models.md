## **Course:** Comptia A+ 220-1201

**Cloud Service Model Overview**

Cloud computing lets you use hosted resources and services instead of running everything on local infrastructure.

- **Cloud deployment models** describe where the cloud runs and who shares it.
- **Cloud service models** describe what the provider manages versus what you manage.
- A cloud can support applications, storage, compute, and networking.

**Deployment Models**

These models focus on ownership, access, and sharing.

- **Public cloud**:
  - A provider hosts the infrastructure.
  - Multiple customers share the same platform.
- **Private cloud**:
  - One organization uses the cloud exclusively.
  - The organization may own or control the infrastructure.
- **Hybrid cloud**:
  - Combines public and private cloud resources.
  - You use it when you need flexibility across environments.
- **Community cloud**:
  - Multiple organizations with shared needs use the same cloud.
  - It often supports common security, compliance, or jurisdiction requirements.

**Service Models**

These models define the level of control and responsibility.

- **Infrastructure as a Service (IaaS)**:
  - The provider supplies core infrastructure.
  - You manage the operating system, applications, and data.
- **Platform as a Service (PaaS)**:
  - The provider supplies the platform for building and deploying apps.
  - You focus on your code and data.
- **Software as a Service (SaaS)**:
  - The provider delivers the application.
  - You use the software without managing the underlying platform.

**Practical Cloud Benefits**

Cloud services support flexible IT operations.

- You can deploy resources quickly.
- You can scale capacity as needed.
- You can place services in different locations.
- You can reduce the need to host everything in your own data center.

**Exam Focus**

For CompTIA A+ 220-1201, you should distinguish deployment models from service models.

- **Deployment models** answer who uses the cloud and how it is shared.
- **Service models** answer what layer the provider manages.
- Match each model to the best business scenario.

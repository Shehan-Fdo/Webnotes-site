## **Course:** Comptia A+ 220-1201

**Memory Basics**

Memory in this context means **Random Access Memory (RAM)**, not long-term storage.

- RAM is **volatile**.
- Data disappears when you power off the system.
- RAM is temporary, high-speed storage for active applications and calculations.
- You must load data into RAM before the system can use it.

**Common Memory Form Factors**

- **Dual Inline Memory Module (DIMM)** is a common desktop memory module.
- **Small Outline Dual Inline Memory Module (SODIMM)** is a common laptop memory module.
- Modern systems commonly use the same general memory slot style.
- Memory specifications still vary by system.

**Memory Technology Terms**

- **Synchronous Dynamic Random Access Memory (SDRAM)** is a memory type used in modern systems.
- **Double Data Rate (DDR)** memory transfers data on more than one clock edge.
- Common DDR generations include `DDR3`, `DDR4`, and `DDR5`.
- You must match the supported memory type for the system.

**Storage Basics**

Storage keeps data when the system has no power.

- Storage is **non-volatile**.
- You use storage for files, applications, and the operating system.
- Common storage devices include hard drives, solid state drives, flash drives, memory cards, and optical drives.

**Hard Disk Drives**

A **Hard Disk Drive (HDD)** uses spinning magnetic platters.

- HDDs store data on magnetic platters.
- HDDs provide random access to stored data.
- HDDs retain data when power is off.
- Mechanical parts limit speed compared with solid state storage.

**Solid State Drives**

A **Solid State Drive (SSD)** stores data without spinning platters.

- SSDs are a common long-term storage option.
- SSDs are faster than traditional hard drives in typical use.
- SSDs have no moving parts.
- SSDs are often used when you want better performance.

**Other Storage Types**

- **Flash memory** is used in portable storage devices.
- **Memory cards** provide removable storage.
- **Optical drives** store and read data with laser-based media.
- **PCIe** storage is also part of modern storage options.
- **Non-Volatile Memory Express (NVMe)** is a storage interface used with fast flash-based drives.
- **M.2** and `mSATA` are storage form factors you may see in modern systems.
- **Serial Attached SCSI (SAS)** is another storage technology you may encounter.

**Practical Distinction**

- Use RAM for active work.
- Use storage for retained data.
- Do not confuse memory capacity with drive capacity.
- Check whether a question refers to temporary memory or permanent storage.

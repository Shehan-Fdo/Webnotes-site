## **Course:** Comptia A+ 220-1201

**Peripherals and Input Devices Overview**

Input devices let you send data into a computer or mobile device.

- Common examples include **mouse**, keyboard, touch screen, scanner, microphone, and biometric devices.
- Mobile accessories can also act as input devices.
- You should match the device to the task and the platform.

**Mouse Devices**

Mice are a common way to control the pointer and enter input.

- Most modern mice connect through `USB` or a wireless link to a `USB` receiver.
- Older systems may use `PS/2` or, on very old systems, serial connections.
- Many current mice use **optical** tracking instead of a mechanical ball.
- Optical mice can have trouble on glass surfaces unless they are designed for glass use.

**Keyboard Devices**

Keyboards remain the most common text input device.

- Modern keyboards usually connect with `USB`.
- Older keyboards may use `PS/2`.
- Some keyboards include extra function keys or special controls.

**Touch and Pen Input**

Touch screens and stylus devices provide direct interaction with the display.

- A **stylus** gives you finer control than a finger on a touch screen.
- Stylus devices often connect through `Bluetooth`.
- Compatibility matters, so you should use a stylus that matches the device.

**Other Input Devices**

Several other peripherals capture data in specialized ways.

- **Scanners** convert physical documents or images into digital form.
- **Microphones** capture audio input.
- **Biometric devices** read physical traits for identification.
- These devices support different workflows, so you should choose the one that fits the use case.

**Mobile Device Accessories**

Mobile peripherals extend input and interaction options.

- Trackpads and drawing pads support precise control.
- Headsets, speakers, and webcams add communication features.
- Docking stations and port replicators expand connectivity for mobile devices.
- Accessory compatibility depends on the device model and connection type.

**Connection and Compatibility Checks**

You should verify the interface before you connect a peripheral.

1. Identify the device type and required connection.
2. Check whether the system uses `USB`, `PS/2`, `Bluetooth`, or another supported interface.
3. Confirm that any adapter or wireless receiver matches the device.
4. Test the peripheral on the target system.
5. Replace or reconfigure the device if it does not respond as expected.

> Use caution with older peripherals and adapters, because connection type and compatibility can limit functionality.

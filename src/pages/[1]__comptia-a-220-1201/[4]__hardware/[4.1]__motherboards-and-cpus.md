## **Course:** Comptia A+ 220-1201

**Motherboard Basics**

The motherboard is the main system board in a computer.

- **Form factor** affects physical size and case fit.
- Standard sizes help you match the board to the case.
- Motherboards vary by desktop, laptop, and mobile device design.
- The board layout determines where major components connect.

**CPU Compatibility**

CPU choice affects the rest of the system.

- **Intel** and AMD motherboards use different CPU designs.
- You cannot swap an AMD CPU into an Intel-based board.
- CPU sockets differ between manufacturers.
- Motherboard and CPU choice often depends on cost and platform needs.

**Memory Slots**

Memory support depends on the motherboard and system size.

- **RAM** stands for random access memory.
- Common slot types include `DIMM`, `SO-DIMM`, and `Micro-DIMM`.
- Slot type usually matches the overall size of the system.
- Memory speed and technology must match the motherboard requirements.

**Motherboard Selection**

You choose the motherboard based on the components you plan to use.

- The board decides which parts are compatible.
- The motherboard type and model affect system capability.
- Standard form factors help you plan the build.
- Compatibility checks prevent mismatched parts.

**Installation and Handling**

Motherboard installation needs careful handling.

- Use **ESD** (electrostatic discharge) protection before touching components.
- An `ESD` mat or wrist strap helps reduce static risk.
- Ground the work surface and case when possible.
- Handle the board carefully because installation can be awkward.

**Troubleshooting Checks**

Use simple checks before replacing parts.

- Confirm the CPU and socket match.
- Confirm the memory type and slot type match.
- Verify the board fits the case and mounting points.
- Check that all required connections are seated.

**Laptop and Mobile Considerations**

Smaller systems often use different motherboard designs.

- Laptop boards can require extra disassembly to access core parts.
- Wireless antenna wires may route through the display assembly.
- Internal layouts are tighter than desktop layouts.
- Service steps vary by model and vendor.

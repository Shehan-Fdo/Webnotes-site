## **Course:** Comptia A+ 220-1201

**Common Network Hardware**

Network hardware helps you connect devices, move traffic, and control access in wired and wireless environments.

- **NIC** stands for Network Interface Card.
- A NIC provides network connectivity for a device.
- A NIC can be wired or wireless.
- Every connected device needs the right type of NIC for the network it uses.

**Routers**

Routers connect networks and direct traffic between them.

- You use a router to link separate networks.
- A router manages traffic flow between networks.
- Routers are common in home and business networks.

**Switches**

Switches expand wired network connections.

- A switch adds more Ethernet ports.
- A switch connects devices on the same local network.
- Managed switches and unmanaged switches both appear in network environments.

**Wireless Access Points**

Access points provide wireless coverage.

- An access point extends Wi-Fi coverage.
- You use access points in larger or multi-floor environments.
- Access points help wireless devices join the network.

**Modems and ONTs**

Modems and Optical Network Terminals (ONTs) connect you to an internet service provider.

- A modem supports cable or DSL service.
- An ONT supports fiber service.
- These devices sit at the edge of the network connection.

**Firewalls**

Firewalls filter network traffic.

- A firewall helps protect the network.
- A firewall allows or blocks traffic based on rules.
- Firewalls are common in home and enterprise setups.

**Patch Panels**

Patch panels help organize cabling.

- A patch panel centralizes cable terminations.
- A patch panel makes cable management cleaner.
- You often see patch panels in larger installations.

**Power over Ethernet**

Power over Ethernet (PoE) sends power and data over one cable.

- PoE reduces the need for separate power adapters.
- PoE supports devices such as access points and other network hardware.
- PoE standards vary by device and deployment needs.

**Practical Device Selection**

You choose network devices based on the connection type and the role you need.

- Use a router to connect networks.
- Use a switch to add wired ports.
- Use an access point to improve wireless coverage.
- Use a modem or ONT to reach the service provider.
- Use a firewall to filter traffic.
- Use a patch panel to organize cabling.

## **Course:** Comptia A+ 220-1201

**Wireless Networking Overview**

Wireless networking uses radio waves to connect devices without cables.

- The IEEE (Institute of Electrical and Electronics Engineers) defines the **802.11** wireless LAN (WLAN) standards.
- The Wi-Fi Alliance uses simpler Wi-Fi generation names for some 802.11 versions.
- Performance varies by standard, band, and the number of connected devices.

**802.11 Naming**

You should recognize both the IEEE name and the Wi-Fi generation label.

- `802.11ac` is also called **Wi-Fi 5**.
- `802.11ax` is also called **Wi-Fi 6**.
- `802.11ax` in the 6 GHz band is called **Wi-Fi 6E**.
- `802.11be` is associated with **Wi-Fi 7**.

**Wireless Bands and Channels**

Wireless standards use different frequency bands.

- `2.4 GHz` is common in older and many legacy-compatible networks.
- `5 GHz` supports higher throughput in many deployments.
- `6 GHz` is used by newer Wi-Fi generations such as Wi-Fi 6E.
- Channel and band selection affect speed, range, and interference.

**Common 802.11 Standards**

Use the standard name to identify the generation and expected capability.

- `802.11a` is an early standard.
- `802.11g` is a later `2.4 GHz` standard.
- `802.11n` is a dual-band standard.
- `802.11ac` improves performance over earlier generations.
- `802.11ax` improves efficiency and capacity in modern networks.
- `802.11be` is a newer generation beyond Wi-Fi 6E.

**Practical Comparison**

| Standard | Common Wi-Fi Name | Key Point |
|---|---|---|
| `802.11a` | Wi-Fi 1 | Early `5 GHz` standard |
| `802.11n` | Wi-Fi 4 | First common dual-band option |
| `802.11ac` | Wi-Fi 5 | Widely used modern standard |
| `802.11ax` | Wi-Fi 6 | Newer efficiency-focused standard |
| `802.11ax` in `6 GHz` | Wi-Fi 6E | Extends Wi-Fi 6 into `6 GHz` |
| `802.11be` | Wi-Fi 7 | Newer generation beyond Wi-Fi 6E |

**Exam Focus**

You should match the standard to the likely scenario.

- Use the Wi-Fi generation label when the question gives a consumer-friendly name.
- Use the IEEE name when the question gives the technical standard.
- Expect questions about frequency bands, relative performance, and standard naming.
- Treat newer standards as generally more capable, but verify the band and device support.

## **Course:** Comptia A+ 220-1201

**Network Type Basics**

Network types are usually categorized by size and location.

- **LAN** means Local Area Network.
- A LAN covers a local area, such as a floor, building, or group of nearby buildings.
- LANs usually provide the fastest local throughput.
- **WAN** means Wide Area Network.
- A WAN connects locations over long distances, such as across cities, states, countries, or worldwide.
- WANs usually have lower throughput than LANs because of distance and transport challenges.

**Common Network Types**

- **PAN** means Personal Area Network.
- A PAN covers a very small area around a person or device.
- **SAN** means Storage Area Network.
- A SAN supports storage-focused network communication.
- **WLAN** means Wireless Local Area Network.
- A WLAN is a local network that uses wireless connectivity, such as 802.11.

**Network Topology Basics**

Network topology describes how devices connect in a network.

- **Physical topology** shows the actual device and cable layout.
- **Logical topology** shows how traffic flows or how the network is organized conceptually.
- You use a physical map to document exact device placement and cable connections.
- You use a logical map to show a higher-level view of the network.

**Common Topology Shapes**

- **Star** topology uses a central device that connects to other stations.
- **Mesh** topology uses multiple interconnections between devices.
- **Ring** topology connects devices in a loop.
- **Bus** topology uses a shared backbone.
- Older Ethernet networks may use coaxial bus layouts, but they are less common in modern networks.

**Practical Documentation**

- Match the map type to the goal.
- Use physical documentation when you need exact placement.
- Use logical documentation when you need a simplified network view.
- Keep topology records current when devices or links change.

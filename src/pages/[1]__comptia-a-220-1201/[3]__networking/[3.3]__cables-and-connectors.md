## **Course:** Comptia A+ 220-1201

**Cable and Connector Basics**

Cables carry data, video, power, and network traffic between devices.

- Choose the cable type that matches the device and interface.
- Match the connector shape before you plug in a cable.
- Use the correct cable length and rating for the job.

**Network Cables**

Network cabling supports wired Ethernet and also connects wireless infrastructure to the rest of the network.

- **Twisted pair** copper cable is common for Ethernet.
- **Unshielded twisted pair (UTP)** is common for indoor use.
- **Shielded twisted pair (STP)** adds more protection against interference.
- **Direct-buried cable** is built for underground or outdoor runs.
- **Plenum-rated cable** is designed for air-handling spaces.

> Use the correct cable type for the environment. Indoor cable is not a safe substitute for outdoor or plenum-rated installations.

**Copper Connector Termination**

You may need to add a connector to a cable during installation or repair.

- Use a **crimper** to attach connectors to copper cable.
- **RJ45** connectors are common on twisted pair network cables.
- The connector pins pierce the insulation to contact the copper conductors.
- Test the finished cable before you install it.

**Peripheral Cables**

Peripheral cables connect common input devices and accessories.

- **USB** stands for Universal Serial Bus.
- USB is the standard connection for keyboards, mice, and many peripherals.
- **USB-C** is a newer reversible connector style.
- **Thunderbolt** supports high-speed peripheral connections.
- **Serial** cables still appear in some legacy equipment.

**Video Cables**

Video cables connect computers to monitors and displays.

- **HDMI** stands for High Definition Multimedia Interface.
- HDMI carries audio and video over one digital connection.
- HDMI uses a 19-pin **Type A** connector.
- **DisplayPort** and **DVI** are also common display interfaces.
- **VGA** is an older analog video connector.
- **USB-C** can carry video on supported devices.

**Storage Cables**

Storage devices often use dedicated internal or external cables.

- **SATA** stands for Serial AT Attachment.
- SATA is common for internal hard drives and SSDs.
- SATA uses separate power and data connectors.
- **eSATA** is the external version of SATA.
- eSATA supports external drive connections over a cable.

**Practical Installation Checks**

You improve reliability when you verify the cable path and connector fit before deployment.

1. Identify the device interface.
2. Select the correct cable type.
3. Confirm the connector matches the port.
4. Attach the connector with the proper tool if needed.
5. Test the cable after installation.

**Troubleshooting Cable Issues**

Bad cabling can prevent devices and applications from working as expected.

- Check for the wrong cable type first.
- Inspect connectors for damage or poor seating.
- Verify that the cable is rated for the environment.
- Replace suspect cables with known-good ones.
- Confirm the link works after replacement.

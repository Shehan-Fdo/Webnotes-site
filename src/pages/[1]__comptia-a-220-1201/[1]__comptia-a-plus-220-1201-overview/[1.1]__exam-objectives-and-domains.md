## **Course:** Comptia A+ 220-1201

**Exam Overview**

CompTIA A+ Core 1 is the first of two exams required for the A+ certification.

- It uses exam series code `220-1201`.
- It is part of version `V15`.
- You must take Core 1 and Core 2 from the same version.
- Core 1 focuses on foundational **hardware**, **networking**, mobile devices, and troubleshooting.
- Passing Core 1 and Core 2 supports entry-level roles like IT support specialist and help desk technician.

**Core 1 Domain Focus**

Core 1 emphasizes practical support skills across several technical areas.

- Install, configure, and maintain hardware components.
- Install, configure, and maintain mobile devices.
- Troubleshoot hardware and connectivity issues.
- Understand networking basics.
- Configure SOHO (small office/home office) networks.
- Explain virtualization concepts.
- Explain cloud computing concepts.
- Use troubleshooting tools to diagnose and resolve technical problems.

**Exam Relationship to Core 2**

Core 1 is only one part of the full A+ certification path.

- You need both Core 1 and Core 2 to earn A+.
- Core 2 covers operating systems, security, software troubleshooting, and operational procedures.
- Core 1 and Core 2 together build a broader foundation for support work.
- The certification also supports progression toward Network+ and Security+.

**Practical Study Implications**

You should align study time with the official exam scope.

- Focus on hands-on hardware and mobile device support.
- Practice basic network setup and troubleshooting.
- Review virtualization and cloud concepts at a high level.
- Use troubleshooting workflows to isolate common issues.
- Keep the exam version consistent when scheduling both tests.

**Key Takeaways**

- Core 1 is the hardware and networking exam in the A+ path.
- The exam version is `V15`.
- The exam code is `220-1201`.
- You must pair it with Core 2 from the same version.
- The objective set centers on support tasks used in entry-level IT roles.

## **Course:** CompTia Security Plus

**Network attack overview**

- Network attacks target communication flows, services, or systems on a network.
- An **on-path attack** is formerly known as a man-in-the-middle attack.
- In an on-path attack, an attacker intercepts communication between a user and a service.
- The attacker can capture sensitive data such as login credentials.
- The attacker can then reuse intercepted credentials to access systems.
- Rogue services can disrupt normal network behavior by answering client requests.
- A rogue DHCP server can hand out invalid or duplicate IP addresses.
- Duplicate addressing can prevent devices from communicating correctly.
- Network attacks often overlap with application attacks and malware delivery.

**On-path attack**

- An on-path attack places the attacker between two communicating parties.
- The attacker can read, alter, or relay traffic.
- This attack is also called **MITM**.
- Credential interception is a common impact.
- Reused credentials can lead to unauthorized access.

**Rogue services**

- DHCP stands for Dynamic Host Configuration Protocol.
- DHCP provides IP address configuration details to clients.
- DHCP has no built-in security in the described material.
- Any device on the network can respond to a DHCP request.
- A rogue DHCP server can cause network misconfiguration.
- Misconfiguration can disrupt internet access and local communication.

**Malware tied to network compromise**

- **Malware** is malicious software that performs harmful functions on a computer or network device.
- A **virus** is an unwanted malicious program that can reproduce itself.
- **Crypto-malware** encrypts files or programs to extort money.
- **Ransomware** denies access to a system or data until payment.
- Ransomware can spread through phishing email or an infected website.
- A **worm** is self-contained and can spread through networks, email, and messages.
- A **trojan** pretends to be a harmless application.
- A **rootkit** can provide full remote access through a backdoor.
- A **keylogger** records keystrokes from an infected machine.
- **Adware** shows ads and pop-ups in a browser.
- **Spyware** installs itself to spy on the infected machine and send stolen information.

**Practical exam focus**

- Match the attack to the observed behavior.
- Look for traffic interception when the scenario mentions credential theft.
- Look for rogue service behavior when the scenario mentions bad IP assignment or DHCP disruption.
- Look for malware spread when the scenario mentions email, websites, or self-replication.
- Use the most specific attack name available from the scenario.

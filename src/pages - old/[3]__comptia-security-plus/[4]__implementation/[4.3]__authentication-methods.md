## **Course:** CompTia Security Plus

**Authentication methods overview**

- Authentication verifies identity before access is granted.
- **Directory services** store usernames, passwords, computers, printers, and other network devices.
- Directory databases are distributed across multiple devices.
- Replication keeps directory data up to date across the environment.
- A single username and password can provide access to multiple resources.
- Microsoft Active Directory is a common directory service.
- Kerberos and LDAP are commonly used to access directory data from external devices.

**Federation and centralized access**

- **Federation** lets separate systems trust each other for authentication.
- Federation supports access across different services without managing separate credentials for each one.
- Use federation when you need centralized identity handling across multiple systems.

**Attestation and device trust**

- **Attestation** helps verify the state or trustworthiness of a device.
- Use attestation as part of the authentication process when device validation matters.
- Treat attestation as an added trust check, not a replacement for credentials.

**Wireless authentication**

- Wireless networks require authentication before access.
- **EAP** stands for Extensible Authentication Protocol.
- EAP is a framework with many methods for authenticating to a network.
- WPA and WPA2 use forms of EAP for wireless authentication.
- Cisco used **LEAP** on early access points that used WEP.
- Cisco later updated authentication to **EAP-FAST**.
- FAST stands for Flexible Authentication via Secure Tunneling.

**Multi-factor authentication**

- **MFA** stands for multi-factor authentication.
- MFA strengthens access control by requiring more than one factor.
- Use MFA to reduce account compromise and credential theft.
- MFA supports defense in depth by adding another layer to login protection.

**Practical study points**

- Match the authentication method to the access scenario.
- Use directory services for centralized credential storage.
- Use federation when systems need shared trust.
- Use EAP-based methods for wireless authentication.
- Use MFA to improve account protection.

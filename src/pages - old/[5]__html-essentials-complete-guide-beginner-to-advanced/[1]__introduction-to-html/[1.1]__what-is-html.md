## **Course:** HTML Essentials: Complete Guide (Beginner to Advanced)

**HTML overview**

- HTML stands for `HyperText Markup Language`.
- It is a **markup language** used to create and structure web pages.
- It defines the meaning and structure of web content.
- Browsers use HTML to display text, images, links, and other elements.
- HTML is the most basic building block of the Web.

**How HTML works**

- HTML uses **elements** to wrap content.
- Elements are written with tags such as `<p>`, `<head>`, and `<body>`.
- Tags usually come in opening and closing pairs.
- Attributes add extra information to elements.
- HTML documents are plaintext files structured with elements.

**What HTML provides**

- HTML gives a page its skeleton or framework.
- It organizes content into headings, paragraphs, lists, images, and links.
- It helps define semantics, or the meaning of content.
- Clear structure supports accessibility and search engine optimization.
- HTML works with CSS for presentation and JavaScript for behavior.

**Core ideas to remember**

- `Hypertext` refers to links between web pages.
- HTML makes the web navigable through linked documents.
- A well-structured document helps browsers handle content more effectively.
- HTML is the starting point for building web pages.
- Modern HTML continues as part of the HTML Living Standard.

**Basic syntax examples**

- A paragraph uses `<p>...</p>`.
- A link uses an element that can point to another page.
- An image uses an element that displays visual content.
- Headings and lists help organize information clearly.
- The `head` and `body` sections separate metadata from visible content.

**Why HTML matters**

- HTML is essential for creating web content.
- It is the foundation you build on before styling or scripting.
- It supports readable, structured, and machine-friendly pages.
- It remains a core skill for anyone learning web development.

## **Course:** HTML Essentials: Complete Guide (Beginner to Advanced)

**Form controls overview**

- Forms and buttons are the main tools for user interaction on a website, along with links.
- Use form controls to collect data or let users change the UI.
- Keep form markup semantic so browsers and assistive tech can interpret it correctly.
- Use labels and correct structure to improve accessibility.

**`<textarea>`**

- Use `<textarea>` for multi-line plain-text input.
- It fits free-form content such as comments or feedback.
- Connect it to a `<label>` with a matching `id` and `for` value.
- Set `name` so the submitted data has a field name.
- Use `rows` and `cols` to control the visible size.
- The `rows` property reflects the visible text lines and defaults to `2`.
- The `HTMLTextAreaElement` interface exposes properties such as `cols`, `defaultValue`, `disabled`, and `form`.
- Use `setSelectionRange()` to set the selection start and end in a textarea.
- The selection state updates immediately, but the highlight appears only when the element is focused.

**`<select>`**

- Use `<select>` to present a menu of options.
- Pair it with a `<label>` for accessibility.
- Include `<option>` elements for each choice.
- Use an empty option when you want a placeholder-like prompt.
- Keep option values clear so submitted data is predictable.

**Buttons and submission**

- Use buttons as core form interaction controls.
- Choose the button type carefully because different button types behave differently.
- Use `name` and `value` on controls when you need submitted data to include them.
- Keep form submission basics simple and consistent.
- Add accessible labels and semantic structure before styling.

**Practical form habits**

- Start with the control that matches the data you need.
- Use a textarea for longer text, a select for fixed choices, and a button for action.
- Prefer short, clear labels.
- Test that the control is usable with keyboard focus.
- Keep advanced or specialized form features for later, need-to-know learning.

## **Course:** HTML Essentials: Complete Guide (Beginner to Advanced)

**Opening links in a new tab**

- Use the `<a>` tag with `href` to create a hyperlink.
- Add `target="_blank"` to open the link in a new browser tab.
- Keep normal internal navigation in the same tab unless you have a clear reason to do otherwise.
- Use new tabs for external sites, documentation, references, or third-party resources when you want to keep the current page open.
- The linked text should still describe the destination clearly.

**Security and behavior**

- `target="_blank"` can create a new browsing context.
- `rel="noopener"` prevents the new page from accessing the opener through `window.opener`.
- MDN notes that `target="_blank"` implicitly provides the same `rel` behavior as `noopener` for `<a>` elements.
- Use `rel="noopener"` when you open untrusted links.
- `noreferrer` hides the referrer information from the destination page.
- `noopener` and `noreferrer` are often used together on external links.

**Practical usage**

- Use a new tab when you want users to compare content without losing the current page.
- Use a new tab when you link to a file or outside resource that supports the current task.
- Avoid new tabs for routine site navigation.
- Test the link in a browser after you add the attributes.
- Confirm that the destination opens as expected and that the current page stays available.

**Core pattern**

- Basic link: `<a href="https://example.com">Visit site</a>`
- New tab link: `<a href="https://example.com" target="_blank">Visit site</a>`
- Safer external link: `<a href="https://example.com" target="_blank" rel="noopener">Visit site</a>`

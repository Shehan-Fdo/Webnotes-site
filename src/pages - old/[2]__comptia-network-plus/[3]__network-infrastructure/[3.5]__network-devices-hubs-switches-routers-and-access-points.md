## **Course:** Comptia Network Plus

**Network device roles**

- A **hub** is a basic Ethernet connectivity device.
- A hub acts as a multi-port repeater.
- A hub repeats incoming traffic to all other ports.
- A hub operates at OSI Layer 1.
- Devices on a hub share the same network segment.
- A hub runs in half-duplex mode.
- A hub does not make forwarding decisions.

**Switch behavior**

- A **switch** works at OSI Layer 2.
- A switch uses the destination **MAC address** to forward frames.
- A switch forwards traffic more selectively than a hub.
- A switch can reduce network traffic.
- Some devices are called switches but also perform Layer 3 routing.
- In this topic, treat a switch as a Layer 2 device unless stated otherwise.

**Router behavior**

- A **router** forwards traffic between different IP subnets.
- A router uses the IP address in the packet to choose the next hop.
- A router operates at OSI Layer 3.
- A router is a Layer 3 device.
- Some home devices combine routing, switching, and wireless access point functions.

**Access point basics**

- A **wireless access point** connects wireless devices to the network.
- A wireless access point is often part of a combined SOHO device.
- SOHO means small office/home office.
- A single device may provide switching, routing, and wireless access point functions together.

**Network interface cards**

- A **network interface card (NIC)** provides network connectivity for a device.
- You find NICs in printers, servers, laptops, and workstations.
- An Ethernet network needs an Ethernet NIC.
- A wireless network needs a wireless NIC.
- A device that connects to multiple network types may need multiple NICs.

**Practical exam points**

- Match the device to the OSI layer it uses.
- Use a hub only as a basic repeater concept.
- Use a switch when you need MAC-based forwarding.
- Use a router when you need subnet-to-subnet forwarding.
- Recognize that consumer routers often combine multiple functions.

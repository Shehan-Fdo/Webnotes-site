## **Course:** Comptia Network Plus

**Firewalls**

- A firewall controls traffic between networks based on rules.
- It helps enforce security policy at network boundaries.
- You place it where you want to filter allowed and denied traffic.
- It can be a hardware device or a software service.
- It is a common network security appliance in enterprise environments.

**Load Balancers**

- A load balancer distributes requests across multiple servers.
- It helps increase capacity and support scalability.
- It is common in data centers and web application environments.
- It receives client requests and forwards them to backend systems.
- It helps avoid sending all traffic to a single server.

**Proxies**

- A proxy acts as an intermediary between a client and a destination service.
- It can receive a request and forward it on the client’s behalf.
- It is used to mediate access to network resources.
- It can support controlled access patterns in network designs.
- It is often discussed alongside load balancers as a traffic-handling device.

**How to Distinguish Them**

- Use a **firewall** when you need traffic filtering and policy enforcement.
- Use a **load balancer** when you need to spread requests across servers.
- Use a **proxy** when you need an intermediary that forwards requests.
- Match the device to the scenario described in the question.
- Focus on the device’s primary function first.

**Exam Focus**

- Know what each device does in a network scenario.
- Know where each device fits in a modern network or data center.
- Expect questions that ask you to choose the best device for a task.
- Remember that these appliances support network performance, access control, and service delivery.

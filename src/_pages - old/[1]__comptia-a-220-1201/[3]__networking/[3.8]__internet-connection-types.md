## **Course:** Comptia A+ 220-1201

**Internet Connection Overview**

Internet access can use several delivery methods, and each one has different speed, cost, and location limits.

- **Satellite** reaches remote sites where terrestrial service is unavailable.
- **Fiber** provides high-speed connectivity over light-based links.
- **Broadband cable** uses the cable provider network for internet access.
- **DSL** uses telephone infrastructure for internet service.
- **Cellular** provides internet through mobile networks.
- **WISP** means Wireless Internet Service Provider and uses fixed wireless links.

**Satellite Internet**

Satellite internet sends traffic from Earth to a satellite and back to Earth.

- It works well for remote locations.
- It is often more expensive than terrestrial options.
- It commonly provides about `100 Mbps` down and `5 Mbps` up.
- It has higher **latency** because the signal travels a long distance.

**Fiber Internet**

Fiber internet uses fiber-optic cabling for data transmission.

- It supports very high speeds.
- It is a strong choice when you need fast, reliable service.
- It is a common modern option for home and business connectivity.

**Broadband Cable**

Cable internet uses the cable television provider network.

- It is widely available in many populated areas.
- It is a practical choice for general home internet use.
- Performance can vary based on local provider capacity.

**DSL**

DSL stands for Digital Subscriber Line.

- It uses existing telephone lines for internet access.
- It is often available where cable or fiber is not present.
- It usually offers lower performance than fiber.

**Cellular Internet**

Cellular internet uses the mobile carrier network.

- It is useful when you need internet on the move.
- It can serve as a backup connection.
- Coverage and speed depend on signal quality and carrier service.

**WISP**

WISP stands for Wireless Internet Service Provider.

- It uses fixed wireless links to deliver internet service.
- It can help in areas without wired broadband.
- It is often used where traditional service is difficult to install.

**Choosing a Connection Type**

You choose the connection type based on location and service needs.

1. Check whether wired service is available.
2. Compare speed requirements with provider options.
3. Consider latency if the application is sensitive to delay.
4. Use satellite or WISP when traditional service is not practical.
5. Use cellular when mobility matters.

**Practical Troubleshooting Notes**

Connection type affects what you check first.

- Verify the service type before you test the device.
- Match expected speed to the connection method.
- Consider distance, coverage, and infrastructure limits.
- Use conservative expectations for remote or wireless links.

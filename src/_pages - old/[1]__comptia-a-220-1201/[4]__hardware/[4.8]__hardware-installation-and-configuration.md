## **Course:** Comptia A+ 220-1201

**Laptop Hardware Overview**

Laptops combine many components into a compact system, so access and replacement vary by model.

- Some laptops allow easy access to memory or storage.
- Other laptops require deeper disassembly to reach internal parts.
- You need precise troubleshooting because laptop components are built to tight specifications.

**Battery Technologies**

Laptop batteries support portable use and come in common rechargeable forms.

- **Lithium-ion (Li-ion)** batteries are common in portable systems.
- **Lithium-ion polymer (LiPo)** batteries are also used in thin devices.
- Battery replacement depends on the model and battery design.
- You should match the correct part before replacement.

**Memory Installation**

Laptop memory is often designed for compact form factors.

- **Small Outline Dual Inline Memory Module (SODIMM)** is a common laptop memory type.
- Some systems use removable memory modules.
- Some systems use soldered memory, which limits upgrade options.
- You should verify compatibility before installation.

**Storage Devices**

Laptops often use storage upgrades to improve performance.

- **Hard Disk Drive (HDD)** storage uses spinning platters.
- **Solid State Drive (SSD)** storage offers faster access and better shock resistance.
- You may replace a hard drive with an SSD during an upgrade.
- You should confirm the storage interface and boot support before migration.

**Keyboard and Key Cap Replacement**

Laptop input devices can be serviceable, but replacement methods vary.

- Some keyboards support part-level replacement.
- Some key caps can be replaced individually.
- You should follow the vendor service process for the specific model.

**Built-in Cameras and Microphones**

Laptops often include integrated communication hardware.

- Built-in cameras support video capture.
- Built-in microphones support audio input.
- You should test these devices before replacing internal parts.

**Near-Field Communication**

Some laptops include **Near-Field Communication (NFC)** support.

- NFC enables short-range wireless communication.
- You should verify whether the device includes NFC before troubleshooting.

**Practical Installation Workflow**

Use a careful process when you install or replace laptop hardware.

1. Identify the exact laptop model.
2. Check the serviceability of the target component.
3. Power down and disconnect the device.
4. Install the replacement part using the correct procedure.
5. Verify that the system recognizes the hardware.

**Troubleshooting Considerations**

Laptop hardware issues often come from access limits or part compatibility.

- Compare the symptom against the installed component.
- Test with known-good parts when possible.
- Use vendor documentation when the device has limited service access.
- Replace only the component that matches the failure.

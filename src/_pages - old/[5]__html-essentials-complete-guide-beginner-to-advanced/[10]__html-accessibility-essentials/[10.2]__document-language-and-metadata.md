## **Course:** HTML Essentials: Complete Guide (Beginner to Advanced)

**Document language**

- Use the `lang` global attribute to define the language of a document or element.
- The value should be a single **BCP 47** language tag, such as `en`, `fr`, or `en-GB`.
- Set `lang` on the root `<html>` element for the page language.
- Use `lang` on smaller elements when content switches languages.
- Leave `lang` unspecified only when you cannot determine the language.
- Avoid `lang=""` because it marks the language as unknown.

**Why language matters**

- Browsers and assistive technologies use language information to interpret text correctly.
- Language metadata helps with pronunciation and text processing.
- Search and styling tools can use language information for language-specific behavior.
- The `lang` attribute reflects the element language, but inherited language may still apply from a parent.

**Metadata basics**

- The `<meta>` element provides document-level metadata.
- Use `<meta>` for information that other elements do not represent.
- Use the `name` attribute for page metadata that applies to the whole document.
- Use the `http-equiv` attribute to simulate HTTP header directives.
- Use the `charset` attribute to declare the document encoding.
- Use the `itemprop` attribute for user-defined metadata.

**Character encoding**

- Declare character encoding early in the document with `<meta charset="...">`.
- Character encoding is essential for correct text display.
- Use a consistent encoding across the document and server output when possible.

**Practical document setup**

- Start with a valid HTML document structure.
- Set the document language on `<html>`.
- Add metadata in `<head>`.
- Include encoding before other content-dependent metadata when possible.
- Keep metadata focused on what browsers and tools need to know.

**Good practice**

- Specify language values explicitly.
- Use accurate language tags for mixed-language content.
- Keep metadata minimal and purposeful.
- Prefer semantic document settings over ad hoc fixes.

## **Course:** HTML Essentials: Complete Guide (Beginner to Advanced)

**Project focus**

- Build a **static website** as a capstone project.
- Use HTML to structure pages and content clearly.
- Apply CSS for styling where needed.
- Keep the site practical and portfolio-ready.

**Core structure**

- Use semantic HTML elements for page layout.
- Include `header`, `nav`, `main`, `section`, and `footer` where they fit.
- Organize content so the page order matches the reading order.
- Use headings to create a clear content hierarchy.

**Accessibility basics**

- Use the right element for the right job.
- Add meaningful `alt` text for images.
- Write clear link text.
- Label form controls when forms are present.
- Use table headers and proper scoping for data tables.
- Keep language simple and avoid unnecessary abbreviations.

**Content patterns**

- Use headings and paragraphs for readable sections.
- Use lists for grouped items.
- Use tables only for structured data.
- Use links for navigation between pages or sections.
- Use images with descriptive attributes when they support the content.

**Classes and IDs**

- Use `class` for reusable styling or grouping.
- Use `id` for unique page targets or anchors.
- Apply CSS selectors to target these attributes.
- Keep naming consistent across the site.

**Project workflow**

1. Plan the site structure and page sections.
2. Write semantic HTML for each page.
3. Add links, images, lists, and tables where needed.
4. Style the pages with CSS.
5. Review accessibility and source order.
6. Test navigation and content clarity.

**Quality checks**

- Confirm the site is easy to scan.
- Check that headings describe each section.
- Verify links work and point to the correct targets.
- Make sure images have useful `alt` text.
- Review tables for clear headers.
- Keep the layout consistent across pages.

**What this capstone reinforces**

- HTML structure for real web pages.
- Semantic markup for accessibility and maintainability.
- Practical use of `class` and `id`.
- Hands-on practice with a complete website build.
- A final project approach that supports portfolio development.

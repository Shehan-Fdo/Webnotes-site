## **Course:** 200-301 CCNA

**Network Security Fundamentals**

- Network security focuses on protecting data and services from unauthorized access and disruption.
- The CCNA security fundamentals domain emphasizes core concepts, mechanisms, and best practices.
- You should understand how to protect **confidentiality**, **integrity**, and **availability**.
- Defense-in-depth uses multiple layers of protection instead of a single control.
- Security policies guide how you protect and manage network resources.

**Threat Landscape**

- Threat actors use tools and techniques to exploit networks.
- Common attack vectors can lead to data loss or service disruption.
- Malware is a major category of threat in network environments.
- You should recognize that vulnerabilities can be targeted by attackers.
- Exploits take advantage of weaknesses in systems, protocols, or configurations.

**Common Security Controls**

- Access control limits who can use network resources.
- **ACLs** (access control lists) help filter traffic based on rules.
- **AAA** stands for authentication, authorization, and accounting.
- Authentication verifies identity before access is granted.
- Authorization defines what an authenticated user can do.
- Accounting tracks user activity for visibility and auditing.

**VPN and Password Protection**

- **VPN** stands for virtual private network.
- IPsec VPNs help secure traffic between sites or remote users.
- Site-to-site VPNs connect networks over an untrusted transport.
- Remote access VPNs connect individual users securely to a network.
- You should secure local passwords on Cisco devices to reduce unauthorized access.

**Protocol and Network Vulnerabilities**

- IP vulnerabilities can be exploited by threat actors.
- TCP and UDP vulnerabilities can also be targeted.
- You should treat protocol weaknesses as part of the overall attack surface.
- Security planning should include mitigation for known vulnerabilities.

**Practical Study Focus**

- Learn the purpose of each control before memorizing terms.
- Match each threat with the control that reduces it.
- Use lab practice to reinforce AAA and password security concepts.
- Review how ACLs, VPNs, and AAA support a layered defense.

## **Course:** Comptia Network Plus

**Security focus**

- Network+ assesses your ability to establish secure networks and mitigate vulnerabilities.
- Security hardening is part of the certification scope.
- You should understand network security fundamentals across devices, services, and protocols.
- The exam also covers security threats, attacks, and vulnerabilities.

**Core security concepts**

- **Confidentiality** protects data from unauthorized disclosure.
- **Integrity** keeps data accurate and unaltered.
- **Availability** keeps systems and data accessible when needed.
- These three goals guide practical security decisions.

**Threats and vulnerabilities**

- A **threat** is a potential cause of harm.
- A **vulnerability** is a weakness that can be exploited.
- Malware is one example of a threat category.
- Social engineering is another common threat category.
- Application attacks are also part of the threat landscape.

**Network device and service protection**

- Firewalls help control traffic between networks.
- Intrusion detection systems help identify suspicious activity.
- Endpoint security helps protect individual devices.
- Secure network administration reduces exposure to unauthorized access.
- Open ports can create risk if you do not need the service.

**Ports and services**

- Every network-based service uses a numbered port.
- Ports range from `0` to `65535`.
- Close unused ports to reduce attack surface.
- Keep only intentionally used services exposed.

**Secure protocols and technologies**

- Use secure protocols when possible.
- `SSL` and `IPSec` are examples of protocols covered in network security fundamentals.
- `HTTPS` is a common secure web protocol.
- `VPN` technologies support protected communication over untrusted networks.

**Wireless security considerations**

- Wireless security includes encryption.
- Wireless security also includes `SSID` management.
- Choose secure settings to reduce unauthorized access.

**Practical study approach**

- Review the objective list before you study.
- Match each objective to a lab or example.
- Focus on real-world security tasks, not memorization alone.
- Use scenario-based practice to connect threats with controls.

**Exam mindset**

- Expect questions about secure design and mitigation.
- Expect questions about device hardening and access control.
- Expect questions about common ports and protocols.
- Expect questions that test your ability to choose the right control for the risk.

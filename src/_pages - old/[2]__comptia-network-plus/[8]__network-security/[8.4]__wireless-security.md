## **Course:** Comptia Network Plus

**Wireless security basics**

- Wireless traffic travels over radio waves, so nearby devices can potentially hear it.
- You should encrypt wireless traffic to keep data from being read in transit.
- Authorized users need the right credentials to send and receive on the network.
- Network security goals include protecting data and reducing vulnerabilities.

**Encryption and authentication**

- **WPA2** is the wireless encryption method identified as the one you should really be using in the referenced material.
- **WPA** and **WPA2** use forms of **EAP** for authentication.
- **EAP** means Extensible Authentication Protocol.
- **LEAP** means Lightweight EAP.
- **EAP-FAST** means Flexible Authentication via Secure Tunneling.
- Authentication can require a username, password, or another access method.

**Access control features**

- **MAC filtering** limits wireless access by hardware address.
- **MAC** means Media Access Control.
- A MAC address identifies a network interface card.
- You must add allowed MAC addresses to the access list.
- New devices and guests require administrative updates to the list.
- MAC-based control is easy to bypass if an attacker learns an allowed MAC address.

**Wireless protection practices**

- Encrypt all traffic on private wireless networks by default.
- Use protocols that support integrity, not just confidentiality.
- Restrict access to authorized users only.
- Review wireless security settings on the access point.
- Consider additional controls such as wireless isolation and geofencing when available.

**Practical administration**

1. Identify the wireless security method in use.
2. Confirm that traffic is encrypted.
3. Verify that authentication is required for access.
4. Check whether MAC filtering is enabled.
5. Update allowed device entries when you add a new client or guest.

**Key takeaways**

- Wireless security addresses the risk of over-the-air interception.
- Encryption protects the contents of wireless packets.
- Authentication controls who can join the network.
- Access control lists can help, but they are not strong on their own.

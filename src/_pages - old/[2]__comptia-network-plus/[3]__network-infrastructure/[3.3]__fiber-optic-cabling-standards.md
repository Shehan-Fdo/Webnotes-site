## **Course:** Comptia Network Plus

**Fiber optic overview**

- Fiber carries network traffic as **light** through glass media.
- It supports long distances better than copper because signal loss is lower.
- It is useful where you need high speed and extended reach.
- It is harder to tap than copper, so it is often used on secure networks.
- Light is not affected by radio frequency interference.

**Fiber types**

- **Single-mode** fiber uses a small core.
- Single-mode is suited for longer distances.
- **Multimode** fiber uses a larger core.
- Multimode is common for shorter links and campus or data center use.
- Choose the fiber type based on distance, bandwidth needs, and budget.

**Multimode standards**

- OM1 and OM2 are older multimode types.
- OM3, OM4, and OM5 are newer multimode types.
- New installations should use OM3, OM4, or OM5 when multimode is required.
- OM3 was designed for laser light sources at `850 nm`.
- OM4 supports longer link lengths than OM3.
- OM5 is a newer wideband multimode option.

**Connector types**

- Fiber connectors must match the connection type you need.
- **SC** means Subscriber Connector.
- SC is a square connector and uses a push-and-lock style.
- **LC** is a common small-form connector.
- **ST** is another common fiber connector type.
- **MPO** is used for multi-fiber connections.

**Practical selection points**

- Use fiber when you need distance beyond typical copper limits.
- Use fiber when you want lower susceptibility to interference.
- Match the connector to the device or patch panel interface.
- Verify the fiber type before you install or terminate cable.
- Use newer multimode standards for new builds when multimode fits the design.

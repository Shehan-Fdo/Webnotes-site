## **Course:** Comptia Network Plus

**Copper cabling basics**

- Copper cabling is the physical foundation of network communication.
- You usually install cabling once, so choose the right cable for the right infrastructure.
- Poor cabling limits reliability and throughput.
- Even wireless networks depend on copper cabling for backhaul to wired devices.
- Common copper cable types include twisted pair and coaxial cable.

**Twisted pair cable**

- Twisted pair is the most common copper cable for wired Ethernet.
- It uses multiple wires twisted together inside a single sheath.
- Twisting helps reduce interference in network links.
- Most Ethernet networking uses twisted pair wiring.
- Balanced power operation uses two wires that interact with each other.

**Cable types and use cases**

- Coaxial cable is another copper media type used in modern networks.
- Twisted pair is the standard choice for most LAN connections.
- Use the cable type that matches the network design and device connection.
- Plan cable placement carefully in ceilings and under desks.
- Rework is difficult after the building is occupied.

**Plenum-rated cable**

- Plenum-rated copper cable is important in air-handling spaces.
- Use plenum-rated cable where building rules require it.
- Cable jacket type affects where you can safely install the cable.
- Match the cable rating to the installation environment.

**Installation and planning**

- Plan cabling before installation starts.
- Treat cabling as a long-term infrastructure decision.
- Install the right cable in the right place the first time.
- Consider future access before you run new wire.
- Wireless access points still need wired connections back to the network.

**Practical study points**

- Know the difference between twisted pair, coaxial, and plenum-rated copper cable.
- Remember that cabling affects both wired and wireless network performance.
- Focus on correct installation, not just cable type.
- Expect limited flexibility after the cable is in place.

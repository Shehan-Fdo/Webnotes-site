## **Course:** Comptia Network Plus

**Cable testing basics**

- Use a **cable tester** to verify continuity between cable ends.
- Check that pin one connects to pin one on the other side.
- Use the test to find missing pins or crossed connections.
- Treat it as a basic continuity check, not a full performance analysis.

**When a cable tester is not enough**

- Use more advanced tools when you need crosstalk or signal-loss analysis.
- Use a **time domain reflectometer (TDR)** for copper cable analysis.
- Use an **optical time domain reflectometer (OTDR)** for fiber analysis.
- Use these tools when you need more detail than a simple tester provides.

**Tone generators and probes**

- Use a **tone generator** to place an analog tone on a wire.
- Use an inductive probe to listen for that tone.
- Use this method to identify one cable in a large bundle.
- Use it when cables run through ceilings, closets, or patch panels.

**Cable crimping and termination**

- Use a **cable crimper** to attach modular connectors to cable ends.
- Use the crimper as the last step after running and preparing the cable.
- Use the correct connector type for the cable and plug style.
- Use crimping when you build or repair patch cables.

**Troubleshooting context**

- Use cable tools when you diagnose wired connectivity problems.
- Look for issues such as bad ports, attenuation, interference, and duplexing problems.
- Use the right tool for the problem instead of relying on one device.
- Document findings, actions, and outcomes after you test and fix the issue.

**Network+ focus**

- Network+ expects you to diagnose and resolve network issues using appropriate tools.
- The certification emphasizes practical networking skills and troubleshooting.
- Cable testing tools support wired deployment, maintenance, and repair.

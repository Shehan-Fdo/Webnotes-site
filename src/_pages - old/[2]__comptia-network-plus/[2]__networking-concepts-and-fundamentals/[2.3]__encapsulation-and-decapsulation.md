## **Course:** Comptia Network Plus

**Encapsulation**

- Encapsulation adds information to data as it moves through network layers.
- The added information usually appears in a **header**.
- A **trailer** can mark the end of a packet.
- The data between the header and trailer is the **payload**.
- Each layer can add its own information before passing data down.
- The OSI (Open Systems Interconnection) model provides the layer structure used to describe this process.

**Protocol Data Units**

- A protocol data unit, or **PDU**, is the unit of data exchanged by a protocol.
- Different layers use different PDUs.
- Ethernet sends data in a **frame**.
- IP (Internet Protocol) uses a **packet**.
- TCP (Transmission Control Protocol) commonly uses a **segment**.
- UDP (User Datagram Protocol) commonly uses a **datagram**.

**How Encapsulation Works**

- Application data starts at the upper layers.
- The transport layer adds its header.
- The network layer adds its own header.
- The data link layer wraps the data for local delivery.
- The physical layer sends the bits across the medium.
- Each layer treats the data from the layer above as payload.

**Decapsulation**

- Decapsulation removes layer information as data moves up the stack.
- The receiving device reads the outer header first.
- Each layer strips off the information it needs to process.
- The device uses the trailer, when present, to help confirm the packet ends correctly.
- The payload moves upward until the application receives the original data.

**Key Exam Points**

- Encapsulation happens on the sender side.
- Decapsulation happens on the receiver side.
- The OSI model is a reference model, not actual hardware or software.
- The model helps you describe how protocols and technologies work together.
- Use the correct PDU term for the layer you are discussing.

**Practical Memory Aid**

- Think of encapsulation as wrapping a package.
- Think of decapsulation as unwrapping that package.
- Headers identify what the data is and where it should go.
- Trailers help indicate the end of the data unit.

## **Course:** Comptia Network Plus

**Subnetting fundamentals**

- IPv4 addresses are 32 bits long and use four 8-bit blocks.
- Each 8-bit block is an **octet**.
- Dotted decimal notation writes IP addresses as decimal numbers separated by dots.
- Each octet ranges from `0` to `255`.
- Subnetting helps you calculate network, broadcast, and host addresses.
- A host is a device with a unique IP address on a network.

**Binary math for subnetting**

- Subnetting relies on **binary math**.
- Binary uses only `0` and `1`.
- Each binary digit is a **bit**.
- Eight bits make one byte or octet.
- You convert between decimal and binary to work out subnet values.
- Binary conversion helps you understand how subnet masks divide addresses.

**Subnet masks and CIDR**

- A subnet mask defines which part of an IPv4 address is the network portion and which part is the host portion.
- **CIDR** (Classless Inter-Domain Routing) notation uses a slash prefix such as `/24`.
- CIDR and subnet masks describe the same addressing structure in different forms.
- Classful subnetting refers to older class A, class B, and class C groupings.
- Modern subnetting gives you more control than classful addressing.

**Common calculation goals**

- Calculate the network address from an IP address and mask.
- Calculate the broadcast address for the subnet.
- Calculate usable host addresses within the subnet.
- Determine how many hosts fit in a subnet.
- Determine how many subnets a mask creates.

**Shortcut methods**

- The **magic number** method reduces the amount of math you need.
- It helps you quickly find subnet boundaries during an exam.
- You can use shortcut methods after you understand the manual process.
- Shortcut methods still depend on knowing the mask and host requirements.

**Practical exam approach**

- Start with the given IP address and subnet mask or CIDR prefix.
- Convert values to binary when you need exact boundaries.
- Identify the network portion and host portion.
- Use the mask to find the broadcast address and usable range.
- Check whether the subnet supports the required number of devices.

**Key terms**

- **Network address**: the first address in a subnet.
- **Broadcast address**: the last address in a subnet.
- **Host address**: an address assigned to a device.
- **Subnet mask**: the value that separates network and host bits.
- **Octet**: an 8-bit block in an IPv4 address.

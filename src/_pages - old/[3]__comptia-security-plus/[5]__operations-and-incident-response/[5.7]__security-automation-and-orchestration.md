## **Course:** CompTia Security Plus

**Security automation and orchestration**

- Objective 4.7 focuses on explaining the importance of automation and orchestration related to secure operations.
- Automation uses scripts or tools to perform repeatable tasks with less manual effort.
- Orchestration coordinates multiple security tools and tasks so they work together.
- SOAR stands for Security Orchestration, Automation, and Response.
- SOAR platforms coordinate, automate, and accelerate security operations tasks.
- These platforms support faster and more consistent incident response.

**Why automation matters**

- Automation can increase productivity.
- Automation can improve resource use.
- Automation can improve product or service quality.
- Automation can improve security and safety.
- Automation helps reduce repetitive manual work in security operations.

**Common use cases**

- Use automation for scripting tasks.
- Use automation for continuous integration and testing.
- Use automation for application programming interfaces, or APIs.
- Use automation for security operations workflows that need consistency.
- Use orchestration to connect separate security tools and share information between them.

**Implementation considerations**

- You do not need to know how to script for the exam.
- Knowing scripting languages such as Python, JavaScript, PowerShell, or Bash helps in IT work.
- Consider how automation affects existing processes before you deploy it.
- Test automated workflows before you rely on them in production.
- Use training and validation to reduce mistakes in automated response.

**Incident response support**

- Automation supports the incident response process.
- Use tabletop exercises to practice response steps.
- Use simulations to validate response readiness.
- Use threat hunting to look for suspicious activity.
- Use root cause analysis to identify why an incident happened.
- Use digital forensics to collect and examine evidence.
- Use investigation data sources to support analysis and response.

**Exam focus**

- Know the difference between automation and orchestration.
- Know that SOAR combines orchestration, automation, and response.
- Know the operational benefits of faster, more consistent security handling.
- Know the main use cases for scripting, APIs, testing, and incident response support.

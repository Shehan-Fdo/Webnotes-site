## **Course:** CompTia Security Plus

**Exam purpose**

- CompTIA Security+ validates baseline skills for core security functions.
- It focuses on practical, hands-on skills for real-world security tasks.
- It supports an IT security career and serves as a springboard to intermediate roles.
- It is widely recognized and used by employers to prove core cybersecurity skills.
- It also aligns with U.S. Department of Defense (DoD) 8140 compliance.

**Core skills covered**

- Identify threats, attacks, and vulnerabilities.
- Include malware, social engineering, and application attacks.
- Detect compromise and understand vulnerability scanning.
- Understand penetration testing concepts.
- Use security technologies and tools.
- Include firewalls, intrusion detection systems, and endpoint security.
- Design secure network architectures.
- Implement secure systems and secure protocols.
- Manage identity and access.
- Cover authentication, authorization, and accounting.
- Assess and manage risk.
- Use risk analysis, mitigation strategies, and business continuity planning.
- Install and configure wireless security settings.
- Implement public key infrastructure (PKI).

**What the exam emphasizes**

- The exam emphasizes current best practices for risk management and risk mitigation.
- It tests practical security readiness and response.
- It reflects skills needed to prevent and address today’s threats.
- It covers securing networks, applications, and devices.
- It supports data integrity, confidentiality, and availability.

**Who benefits from it**

- It suits people building a cybersecurity foundation.
- It helps help desk technicians moving toward security work.
- It supports general IT professionals who perform security tasks.
- It is relevant for network administrators, security administrators, and IT auditors.

**Exam format**

- You earn the certification by passing one exam.
- The exam includes multiple-choice questions.
- The exam includes performance-based questions.

**Study focus**

- Learn broad security concepts before narrowing to a specialty.
- Practice scanning and assessing networks for vulnerabilities.
- Review secure access control concepts.
- Review secure architecture and system design.
- Review risk management and business impact.
- Review wireless security and PKI.

**Practical takeaway**

- Treat Security+ as a baseline security certification.
- Focus on applied knowledge, not memorization alone.
- Build comfort with common security tools and concepts.
- Connect each topic to real operational security tasks.
